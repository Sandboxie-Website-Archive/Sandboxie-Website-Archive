User created templates for various programs including MBAE, MailBrid, Cyberfox & many more. Usage and Download: See Forum Topic: http://forums.sandboxie.com/phpBB3/viewtopic.php?f=22&t=22319

Sandboxie Portable

Contributed by wraithdu, a member of the Sandboxie forum.

The SandboxiePortable package guides you in creating a Sandboxie installation that you can take with you on removable media and use on other computers. It keeps all your sandbox data and settings on the media with it.

Note: You must have Administrator privileges to use SandboxiePortable. Due to licensing restrictions, SandboxiePortable cannot take your registration data with it.

Usage and Download: See Forum Topic: http://forums.sandboxie.com/phpBB3/viewtopic.php?f=22&t=2299

Block Process Access

Contributed by wraithdu, a member of the Sandboxie forum.

An add-on DLL to block sandboxed processes from reading the memory of unsandboxed processes. They are still allowed to read memory from processes running in their own sandbox however.

Note: You'll need the Microsoft Visual C++ 2008 SP1 Runtimes for the DLL to work.

Usage and Download: See Forum Topic: https://forums.sandboxie.com/phpBB3/viewtopic.php?t=4885

Sandboxie Delete - Delete multiple sandboxes

Contributed by wraithdu, a member of the Sandboxie forum.

Small utility to automate the termination and emptying of multiple sandboxes at once. See the README for instructions on usage. As an added bonus, it also cleans old (and no longer relevant) BoxExpandedView settings in the Sandboxie Ini configuration file.

Usage and Download: See Forum Topic: https://forums.sandboxie.com/phpBB3/viewtopic.php?t=4462

Sandboxie Shortcut Creator

Contributed by wraithdu, a member of the Sandboxie forum.

Small utility to create shortcuts that start any program sandboxed. Can be considered complementary to the Add Shortcut Icons command in Sandboxie Control.

Usage and Download: See Forum Topic: https://forums.sandboxie.com/phpBB3/viewtopic.php?t=1983

BorderGuard - Display borders around sandboxed windows

Contributed by Micahs, a member of the Sandboxie forum.

Paints borders around sandboxed windows. Unlike the border settings in Sandbox Settings > Appearance, can paint a border around more than one window at a time.

Usage and Download: See Forum Topic: https://forums.sandboxie.com/phpBB3/viewtopic.php?t=3208

SandboxDiff - List differences in registry and files

Contributed by majoMo, a member of the Sandboxie forum.

Usage and Download: See Forum Topic: https://forums.sandboxie.com/phpBB3/viewtopic.php?t=3606

SandDiff - List differences in registry and files

Contributed by Buster, a member of the Sandboxie forum.

Compares the contents of the sandbox to the rest of the system, and lists the differences. Typically useful after installing software into the sandbox.

Usage and Download: See Forum Topic: https://forums.sandboxie.com/phpBB3/viewtopic.php?t=6312

Buster Sandbox Analyzer

Contributed by Buster, a member of the Sandboxie forum.

Buster Sandbox Analyzer is a tool designed to analyze the behaviour of sandboxed processes and determine if they are malware.

Usage and Download: See Forum Topic: https://forums.sandboxie.com/phpBB3/viewtopic.php?t=6557
