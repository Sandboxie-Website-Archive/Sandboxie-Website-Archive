Re: Uninstalling programs installed inside sandbox

I assume that you know to install programs inside of a sandbox by...
1) Create a sandbox to use for the program (Sandbox > Create New Sandbox).
2) Right-click the program's installer file, and then click on "Run Sandboxed".
Choose the sandbox which will contain the program, from the list.
