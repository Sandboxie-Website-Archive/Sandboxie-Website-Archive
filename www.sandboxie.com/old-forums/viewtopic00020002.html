Adobe Reader DC Quick Recovery [SOLVED]

Posted: Wed Jul 26, 2017 9:08 am
by Needaname
Hello everybody.

Problem:
If you open a .pdf file with Adobe Reader DC from Firefox or MS Office Programs which run inside Sandboxie, there is an issue with saving the .pdf file.
If you try to save it to a quick recovery folder the quick recovery message pops up, but after clicking any of the recoverymethods adobe reader gives an error message stating that the program is already running. The file wont be saved.

Workaround:
You have to close adobe reader, then select the file inside the quick recovery pop up window and recover it. Now the file is recoverd without any errormessages.

Tested with FF 54.0.1, Outlook 2010 and Adobe Reader DC Version 2017.009.20044 on Win7 64Bit Professional.
SB-Version 5.20 (registered/active license)

It seems to only behave that way with Adobe Reader DC. I also tried SumatraPDF (v3.1.1 64-Bit) without any issues.
