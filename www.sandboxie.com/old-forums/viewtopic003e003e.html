Sandboxie version 2.79 Released

Posted: Tue Feb 27, 2007 2:00 pm
by tzuk
Download version 2.79.5

Download 64-bit Sandboxie version 2.79.5

* New Sandboxie Control command to remove sandbox

* New Sandboxie Control settings to expose only Firefox bookmarks (rather than the entire profile folder), and IE favorites

* New Sandboxie Start command line option: /box:current to run program in currently selected sandbox

* Auto-delete slightly adjusted to make more sense

* AutoImportReg changed to a more generic AutoExec
** To emulate AutoImportReg behavior, specify: AutoExec=regedit /s path\to\file.reg

* Functionality of sandboxed ProtectedStorage enhanced
