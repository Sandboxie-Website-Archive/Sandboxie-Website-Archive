well for the first issue, just open up sandboxie control, goto Configuration -> Edit Configuration and add either this line:
OpenFilePath=%HOMEPATH%\Application Data\Opera\Opera\profile\opera6.adr
that way when you save bookmarks, they will get saved to the real bookmark file

or this line: RecoverFolder=%HOMEPATH%\Application Data\Opera\Opera\profile\opera6.adr
that way you can recover that file before you delete the sandbox.

now the second issue im not sure what you mean? are you saying that you cant print pages from opera when it is sandboxed, but you can print pages when opera isnt sandboxed? if thats the case, it may have something to do with your printer (some ppl have had printing problems lately) in which case you would have to run a trace to see whats wrong.

ok try this (for your first issue):

open opera (sandboxed or unsandboxed) and goto Help -> About Opera then goto the path's section and look for the bookmark path, it should be "C:\Documents and Settings\yournamehere\Application Data\Opera\Opera\profile\opera6.adr" (except instead of saying yournamehere, it should say whatever your windows account name is) well that is the path to your bookmark file, so all you would have to do is open the path by doing this:

OpenFilePath=Opera.exe,C:\Documents and Settings\yournamehere\Application Data\Opera\Opera\profile\opera6.adr

just remember to replace yournamehere with w/e your account name is.

also make sure you place it in the right place in your ini file (which would be under "Default Box" on its own line, you'll know you are looking in the right place when you see a bunch of other openfilepaths and openkeypaths), now the print issue, you would have to trace to find out what might be blocked, well i hope that helps you this time.
