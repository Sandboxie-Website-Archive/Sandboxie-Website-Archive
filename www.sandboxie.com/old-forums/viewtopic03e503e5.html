Minecraft.exe is the Launcher which updates the game files.
https://minecraft.net/download
https://s3.amazonaws.com/Minecraft.Down ... ecraft.exe

Minecraft Launcher needs JRE, a link shall open in your browser for Java Setup, which I installed also in the Sandbox.

I sent my Minecraft Log In details in privmsg, which upon entering in the Launcher reveals the Play button.

To reproduce the problem:
As I'm not sure what the fullscreen/windowed toggle key is for Minecraft, just hit Esc (in game window) and find the FULLSCREEN option in the rendered GUI. It will crash going fullscreen and, then crash upon every start (itself trying fullscreen). Sbie 3.76 passes no problem.
