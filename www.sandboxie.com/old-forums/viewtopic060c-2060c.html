Hello jg4545,

https://www.sandboxie.com/SBIE2205
https://www.sandboxie.com/SBIE2313

Adobe Acrobat installs a printer driver, among other things.
If a program is trying to install a driver or service, it will not be able to install inside Sandboxie.

Try installing the application on your host, and then running it Sandboxed.
