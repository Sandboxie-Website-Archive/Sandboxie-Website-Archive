In my view, you're better off not copying the entire configuration file from the old computer.
The [GlobalSettings] settings will likely be incorrect unless you have identical programs installed on both computers.

There's a [UserSettings] section in the sandboxie.ini file that records the user name (Windows logon name) that you use on the computer.
[UserSettings_XXXXXXXX]

SbieCtrl_UserName=xxxxxxx

If your logon name on the new computer is different from the logon name used on the other computer, you will likely wind up with 2 [UserSettings...] sections after you use your previous configuration file with Sandboxie on the new computer.

Except for those two sections, the rest of the file should be OK to copy into sandboxie.ini on the new computer.
If you don't use Configuration > Edit Configuration to paste lines into the new computer's configuration file, be sure to use Sandboxie Control's menu:
Configuration > Reload Configuration
after the editing is done.

Gator, the Sandboxie.ini just holds the settings. It's a configuration file. If you move it to a different computer and load it in Sandboxie, it will give you the same settings you had on the computer you copied the file from.

You do not have to move empty sandbox folders, those are created when you run an application in a sandbox.

The only folders you have to move are those that contain applications installations or modified files. You just copy those folders to the same location on the other computers, or if you change their location, you need to edit the settings file as well to let it know of the sandbox's new location.
