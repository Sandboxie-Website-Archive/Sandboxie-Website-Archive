Hello TryKindness,

To change the rooth path, open Sandboxie Control --> Sandbox ---> Set Container Folder, choose the desired location.
This is what it looks line under Global Settings in Sandboxie.ini
[ GlobalSettings]
FileRootPath=C:\Sandbox\%USER%\%SANDBOX%

As for your second question:
https://www.sandboxie.com/index.php?SandboxieIni
Sandboxie looks for the file Sandboxie.ini in the following folders, in this order:
In the Windows folder: C:\WINDOWS on most Windows installation; C:\WINNT on Windows 2000
In the Sandboxie installation folder: Typically C:\Program Files\Sandboxie.
...
Sandboxie does not support any other, custom location for the Sandboxie.ini file.
