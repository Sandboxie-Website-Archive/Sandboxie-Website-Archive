Re: different default programs per sandbox

Posted: Tue Aug 23, 2016 8:58 am
by Craig@Invincea
You can force any program (in the paid version of SBIE) to automatically open in a SB when run.

However, Sandboxie opens the default program depending on what you have set as the default program in Windows (such as a PDF reader.)

That's handled by Windows, not SBIE.

No. Windows still runs default programs. Sandboxie is simply a Folder. A container. A Sandbox. Setting default programs are handled by the OS.

You might want to look at a VM.
