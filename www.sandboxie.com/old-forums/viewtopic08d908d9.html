Post by Craig@Invincea » Fri Oct 16, 2015 9:16 am

Users running Windows 10 may still see some "random" Chrome crashing. This is not related to any KB. It's something we have been looking at. Thus far, we cannot reproduce it accurately. However, if you are getting random Chrome crashing, please post information on hardware you are running and if possible, please run this PowerShell command and create a txt file for programs instalalled so we can take a look.

Powershell command (run as admin)
Get-WmiObject -Class Win32_Product | Select-Object -Property Name > C:\Software\PCapps.txt

(for PCapps.txt, you can name it whatever you like as long as it ends in .txt
