Hello Creave,

Here's how you can manually open Firefox outside Sandboxie in order to update it (while still having it set to Forced in a Sandbox):
Right Click on the program --> Run Sandboxed --> Run outside Sandbox
OR
Right click on the Sandboxie tray icon --> Disable Forced Programs
OR
Sbie Control --> File --> Disable Forced Programs (this allows you to set the amount of time that you want them disabled for).

Regards,
