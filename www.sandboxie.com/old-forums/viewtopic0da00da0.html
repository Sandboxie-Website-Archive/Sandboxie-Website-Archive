Sandboxie Support Update

Posted: Thu Oct 12, 2017 2:02 pm
by Barb@Invincea
Greetings all,

We are making some changes regarding how are going to be addressing Sandboxie issues for the time being.

Ideally, we are aiming at posting a new Beta or Release version every month.
Please find the beta versions here

Every post will receive a tech support answer as usual, and it will be troubleshot as much as possible in order to find a workaround/solution. If no solution is found at the tech level, the problem will get escalated to development and will be addressed based on its category (as provided below).

1 – Primary Focus: OS problems / Security bypasses. Any issue that prevents Sandboxie from properly working on a given OS (XP being at the bottom of the list, and Windows 10 at the top. Vista is not officially supported), or any security issues will be prioritized.

2 –Secondary Focus: Major applications -
Browsers -> Firefox, IE, Chrome (other browsers will be considered based on issue severity/dev availability).
MS Office Applications -> Word, Excel, Outlook (other Office applications based on issue severity).
To an extent, issues with Antivirus Software that cannot be fixed using a workaround. (See this thread & the Known Conflicts Section for AV compatibility).

3 – Anything else, including Feature Requests, will be reviewed and logged as usual. However, no ETA will be provided for a fix. This means that we will be aware of the issue, we will try to provide technical support as much as possible, but if the problem requires dev time it may not receive a solution right away.

We thank everybody for your patience and support.
