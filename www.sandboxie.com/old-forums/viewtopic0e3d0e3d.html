by tzuk
In Windows 8.1 the component that catches crashes and does Windows Error Reporting is working in a way that Sandboxie doesn't support.

The message "SBIE2205 Service not implemented: NtCreateProcessEx" refers to that unsupported functionality.

It is not a priority to add this functionality because the underlying problem is that the program has crashed for some reason.

So my question to recent posters:

mystique - What is Microsoft Pinyin IME ?

Orbit and xsilon - What programs are you trying to run?
