Hello Sickest,

It looks like your post is not related to the original one, so I am going to move this to Quick Questions.

Regarding your questions:
Can you enable admin mode through one the batches?
Please see https://www.sandboxie.com/StartCommandLine and look for elevate.
If you, tell sandboxie to delete contents of a sandbox, it doesn't really delete the contents.
Right-click on your Sandbox ---> Delete contents.
If that doesn't work, please disable any Security Software and re-try the process.
If you want to manually delete your Sandboxes via Windows explorer, they are typically located under C:\Sandbox .
For example after I delete the contents of the sandbox, uninstalled sandboxie with the option to not save preferences. I reinstalled the software and crated the same sandbox as before with the same name... and try to open a web browser it has the same saved passwords and cache from before.
This has to be a huge glitch.
If you didn't delete the contents of your Sandbox, and only uninstalled/reinstalled it, the contents inside the Sandbox will remain the same.
Also, for the cache portion:
Is the cache outside Sandboxie clear?If not, Sandboxie will pick it up from your host browsers.
Another option is that you are logging in to your browser, in which case is your account passing your settings.
Please see this thread for more information:
viewtopic.php?p=74194#p74194

Regards,

Hello Sickest,
The problem is, it doesn't actually delete the contents
So, if you navigate to your Sandbox location, are there contents inside each Sandbox? Try manually deleting them as suggested before, which will remove them directly via Windows Explorer.
You can test creating a new file inside the Sandbox, do not recover it, then delete the contents and navigate to the folder location to see if the file is still there or not.

If after that you are still seeing cached passwords, then you are either logged in to the browser, or they are cached in your host computer (or perhaps you are using an extension?). Are you using the latest Sandboxie beta?
Also, make sure you do not have any templates activated for your Web Browsers. If you post your Sandboxie configuration file, I will have a look --> Configure --> Edit Configuration
Paste the output here
Highlight the output and hit the "</>" button in the forum so it is formatted.

Regarding Steam, that's not related to deleting contents, it is some sort of issue with Steam itself and Sandboxie. And, as explained, running it as Administrator is required for it to work. You can try the command I provided or you can right-click on Steam and use the Compatibility settings to run it as Administrator, see if that works for your setup.

Regards,
Barb.-
