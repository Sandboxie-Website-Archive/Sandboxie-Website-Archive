To an anti-virus program that's running in memory, the sandboxes look just the same as any other folder.
So, if some malware should show up in a sandbox, it will be detected the same as if it was in any other folder.

Some AV programs install a toolbar (unsandboxed), and special settings would be needed in Sandboxie if communications from the sandboxed program to that toolbar is desired.
