You could do a couple of things.

1. Make Chrome your default browser and create a sandboxed shortcut to run Firefox when you want to. You are going to create a new sandbox
and you ll have Firefox run in the new sandbox and continue running Chrome in the DefaultBox.

First, you need to create a new sandbox for Firefox. You can do that via Sandbox>Create new sandbox.You can name it Firefox.

And create the sandboxed shortcut via Sandboxie control>Configure>Windows shell integration, click Add shortcut icons, Select the new sandbox, look for Firefox in one of the menus and click it. After the click, you ll find the shortcut at the desktop. If you click the shortcut, Firefox will run in the new sandbox.

And when you click the SandboxedWebBrowser icon, Chrome will run in the DefaultBox.

Remember, if you are using Sandboxies free version, you can only use one sandbox at a time.

2. Another thing you could do would be to buy a Sandboxie license and force Firefox and Chrome to run sandboxed automatically in their own sandbox.
