Re: Firefox crashes after downloading files

Post by Bertus » Sat Dec 19, 2015 7:27 am

I solved my problem with Firefox crashing after downloading files.
After some experimentation with updating drivers I could not fix the problem. I finally fixed it by enabling windows DEP (Data Execution Prevention) for all programs and services. By default Windows only enables DEP for windows programs and sevices.
For me this fixes the problem for Firefox, Palemoon and Cyberfox.
