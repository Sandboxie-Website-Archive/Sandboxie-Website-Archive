i'm using clean slate on a few config's of my own, it creates a layer over the existing system, when enabled it will capture all changes in it's layer instead of the 'real' system. so upon next reboot you're back with a clean slate

it's just a click of a button to disable the layer and make changes to the real system. then enable it back.
it's fully compatible with all kinds of software you want to allow changes or select unlayered folders etc.

tried deepfreeze but couldn't get it to work on windows server 2003 so i dropped it. same functionality though!
