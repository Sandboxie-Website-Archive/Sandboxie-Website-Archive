Hello Mings and all,

I have been testing disabling different AVG components and changing options, however I was not able to get it to work until I added the following to the Sandbox settings:

Right Click on your Sandbox --> Sandbox Settings
Resource Access -->File Access --> Write-Only Access
Hit Add and enter the following:
C:\Program Files (x86)\AVG\Antivirus*

After that, I was able to launch Firefox and Chrome in the Sandbox. (I have not tested other apps yet).
Still looking into this, will update this thread with further info, whenever possible.

Regards,
Barb.-

Hello HolmesIV,

If you have AVG installed, the steps above should cover the issue. Be sure to test them in a new Sandbox with default settings (you can also try deleting the contents of your current one, to see if that helps).
If you are using a different antivirus, please let us know which one, I believe your last post mentioned BitDefender, we have a specific fix for that one:

Free version:
viewtopic.php?f=11&t=21539&start=165#p128854
If you have the paid version, the workaround is the same, except for:
WriteFilePath=C:\Program Files\Bitdefender\* (instead of WriteFilePath=C:\Program Files\Bitdefender Antivirus Free\*)

If you are still experiencing problems, be sure to have fully uninstalled your previous antivirus software (many provide a removal tool, as there might be leftovers), then reboot and test the steps in a new sandbox with default settings.

Please provide this information should the problem persist:
