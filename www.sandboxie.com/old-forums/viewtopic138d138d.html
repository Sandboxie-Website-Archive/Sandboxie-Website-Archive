It's probably controlled by the setting in sandboxie.ini:

[UserSettings_.......]

...
SbieCtrl_ShouldDeleteNotify=n
...
...

To change the setting to "n" as shown above, use Sandboxie Control > Configure > Edit Configuration
so that a Reload Configuration will automatically occur, and put the change into effect.
