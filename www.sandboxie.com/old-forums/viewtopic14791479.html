Changing Sandboxie.ini save location.

Posted: Mon Apr 14, 2008 6:19 am
by sev
Is there any way to change the location of Sandboxie.ini settings file?

I'm sure there's a lot of people like me who like their programs as portable as possible and I, for one, would really appreciate this option.

Posted: Mon Apr 14, 2008 9:43 am
by wraithdu
You can place your INI file in the program directory if you want to. Best way to do this if you have SB running with the INI in the default Windows location -

1. shutdown SbieCtrl and SbieSvc (net stop SbieSvc)
2. move your INI file to the program directory
3. open a cmd prompt in your Sandboxie directory and run these commands -

a. start.exe /reload (not to be confused with the windows 'start' command)
b. net start SbieSvc
c. SbieCtrl.exe

All done
