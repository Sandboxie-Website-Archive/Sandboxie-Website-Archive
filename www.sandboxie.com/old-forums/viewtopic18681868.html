UI Suggestion for Quick/Immediate Recovery Dialog Box

Posted: Tue Aug 16, 2016 9:02 am
by Tridens
I notice that when there is more than one file in the Quick (or Immediate) Recovery Dialog box, that after clicking "Recover", the file selection in the upper box doesn't automatically move to the next available file...the selection just disappears. While not a big deal, this requires moving the mouse to the upper box, selecting the next file, then moving back to the "Recover" button. With a long list of files this takes a while.

Please move the selection in the upper box automatically to the next file in the list so one can just repeatedly click "Recover" (or press Enter key) when moving multiple files from Quick/Immediate Recovery.

Thanks!
