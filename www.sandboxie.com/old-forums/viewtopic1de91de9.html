so i run my games in this sandbox

[Games]
Enabled=y
ConfigLevel=4
AutoRecover=n
AutoRecoverIgnore=.jc!
AutoRecoverIgnore=.part
RecoverFolder=%Favorites%
RecoverFolder=%Personal%
RecoverFolder=%Desktop%
LingerProcess=trustedinstaller.exe
LingerProcess=wuauclt.exe
LingerProcess=devldr32.exe
LingerProcess=syncor.exe
LingerProcess=jusched.exe
LingerProcess=acrord32.exe

ForceFolder=D:\Games
OpenFilePath=D:\Games\
OpenFilePath=pes6.exe,%Personal%\KONAMI\
OpenKeyPath=pes6.exe,HKEY_LOCAL_MACHINE\SOFTWARE\KONAMIPES6
OpenKeyPath=lemmings.exe,HKEY_LOCAL_MACHINE\Software\Visual Sciences
OpenIpcPath=\BaseNamedObjects\ATITool*
OpenIpcPath=\BaseNamedObjects\SnagIt*
OpenWinClass=ATIToolClass
OpenWinClass=SnagIT5UI
NeverDelete=y

so i force the path to games folder and open it so they can save the savegame files correctly.
Some games save the savegame files (like pro evolution soccer 6 (pes6.exe) or GTA) in my documents so i open the path of my documnets to them
Also if you want to take screenshots with snagit you one the Ipc an winclass.
I also use ati tool (gpu oveclocking utility) and i've set it do overclock when it detects 3d game. To detect properly in sandbox, i open one ipc and one winclass.
Also i set autorecover to no as you dont wand sanboxie to pop up while gaming.
