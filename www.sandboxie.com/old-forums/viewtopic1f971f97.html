wraithdu wrote:That's strange, I tested Sandboxie with the %SystemDrive% and %ALLUSERSPROFILE% tokens which are evironment variables only (not Shell Folders) and it worked fine.

If you open a cmd.exe prompt and just type 'set' do you see your %RootDrv% variable and expansion? If so, then you might want to post this in the Sandboxie problems forum and wait for an answer from tzuk.
Control.exe is run in the user's context. SbieSvc.exe is run in the context of the NT\AUTHORITY system. ALLUSERPROFILE and SystemDrive are defined in both contexts. But none of the user's own environment variables are defined in NT\AUTHORITY. That explains why you succeeded and why I didn't.
