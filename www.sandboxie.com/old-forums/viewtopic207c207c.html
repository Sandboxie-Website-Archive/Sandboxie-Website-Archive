I like the way Guest10 describes on this recent thread what happens to registry changes. Read the top half of his post. I think it answers your questions about the registry and Sandboxie.
http://www.sandboxie.com/phpbb/viewtopi ... ht=reghive

To recover all files that are available for recovery all at once, select the "Select all" option that can be seen at the top right corner of the Recovery window. You cant recover programs that are installed in a sandbox that way. If you want a program that you tried in a sandbox to go to your real system, I think its best to just do the installation as they are normally done. I know people can move folders of programs installed in a sandbox out of the sandbox but I never done that or know if it does work OK.

If you like to see the files that are created by programs that are running sandboxed or that you install in a sandbox, navigate to: Sandboxie control>View>Files and folders,

The only thing that you really have to know about settings in order to install a program in a sandbox is that you have to leave Restrictions (Sandbox settings>Restrictions) as they come by default. Most programs can be installed sandboxed but not all.
