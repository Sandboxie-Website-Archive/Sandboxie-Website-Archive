Combined 32/64 installer:
http://www.sandboxie.com/SandboxieInstall.exe

Separate:
http://www.sandboxie.com/SandboxieInstall32.exe
http://www.sandboxie.com/SandboxieInstall64.exe

Changes in 5.12 RC
1) QWaveDrv pipe (google hangouts) is set to open by default.

Changes in 5.11.11
1) Another fix for Chrome 52.

Changes in 5.11.10
1) Fixed rare printing crash.

Changes in 5.11.9
1) Fixed default browser issue.
2) Chrome 52 now works.

Changes in 5.11.8
1) Fixed 2 Excel clipboard related crashes
2) Environment variable USERNAME now shows proper username instead of "SYSTEM" in sandboxed applications.
3) The new, faster loader code developed for Windows 10 is now used for Win 8+.

Changes in 5.11.7
1) Fixed audio problem with MPC, etc. introduced in 5.11.6

Changes in 5.11.6
1) Fixed Win 10 FR 14328 issue with WOW64 applications crashing.
2) Sandboxie now automatically restarts MS Office ClickToRun service during installs/upgrades.

Changes in 5.11.5
1) Fixed Win 10 FR 14316 BSOD.
2) Fixed MS Office ClickToRun performance problem caused by missing '*' on line OpenIpcPath=\RPC Control\ClickToRun_Pipeline* in templates.ini.

Changes in 5.11.4
1) Fixed Win 10 FR 14316 (SBIE1113 Cannot find Nt system service, reason MASTER TABLE).
2) MS Office 2016/365 ClickToRun is now supported. See http://forums.sandboxie.com/phpBB3/view ... 11&t=22645
3) Added qWave driver (Google Hangouts) to templates.ini under Applications->Miscellaneous

Changes in 5.11.3
1) Fixed Win 10 FR 14295 crash in Chrome & FF.
2) Sandboxed applications are no longer allowed to change the system power configuration.

Changes in 5.11.2
1) Fixed rare printing crash in Chrome-32 under Win 10-64.

Changes in 5.11.1
1) Fixed Win 10 FR Build 14279 (SBIE1113 Cannot find Nt system service, reason MASTER TABLE)
