Hello LordTyrion,

You can find more info about error 12029 here, it may help you determine what's blocked:
https://msdn.microsoft.com/en-us/librar ... s.85).aspx

I tested disabling TLS 1.2 (looks like the default ones now are TLS 1.0, 1.1 and 1.2) and received the exact same message you did. Try enabling that one (you can use my previous post as a guide).

Another option is Restoring settings:
Run inetcpl.cpl
Go to "Advanced" tab
Hit "Restore Advanced Settings" (This should enable the TLS settings as needed).
Apply and Ok
Re-test activation

Regards,
