I don't even allow SRT exes to have run access in any of my chrome boxes. It's literally what the name implies in that it attempts to detect and aid in 'reporting' (sharing a list of) the software you have installed on the system. It's also supposedly involved in some cleanup stages of chrome as related to uninstalls but if is all kept in the sandbox anyhow there is a 99.999999999999999999999999999% chance you don't even need to let it run to start with, let alone bother worrying about if it shouldn't be listed as a lingering process.
