SBIE v3.76 is not supported. It lacks true x64 support. It's long since deprecated. Only SBIE v5+ works with Win 10.

v3.76 will not work with MS Windows after (if not before) Oct 2015 because Windows updates have broken SBIE. It lacks updates for 2+ years.

As for games, we recommend games be installed on your host machine, and then "run as sandboxed."
Not all games can run directly in the SB, this is even more of the situation as you go from 7 to 8.1 to Win 10 as MS makes it even more and more difficult for programs to run directly as sandboxed.

We focus on browsers / attachments etc to be directly installed into a SB. Especially in Win 10.

As for programs lingering. In v5.10 (and previous beta's) have addressed that. However, it's no uncommon for them to hang around for several seconds to several minutes per the Dev that is working on that part of the code. The "work" arounds in old version of SB seemed to close program quickly, but that was a "side affect" and the correct behavior with the new code to support Windows updates and Windows 10 works as it should.
