Post by tzuk » Tue Nov 08, 2011 5:08 pm

As suggested earlier in this topic, a good starting point is a new sandbox with default configuration. You can install Origin + BF3 into this sandbox or you can have them installed on the computer (outside the sandbox) and just use Run Sandboxed on the shortcut for the game.

When you confirm this works, start adding Blocked File Access settings to block off access to specific folder locations and then check if the game still works fine. If you review this topic, some of the comments say that blocking too much will cause the game to fail to load, so this is something to take into consideration.
