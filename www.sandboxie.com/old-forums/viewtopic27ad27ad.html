Post by btm » Wed Nov 11, 2015 9:07 am

It could be because you have the sandbox set to force a program, eg game.exe
It could also be a forced folder 'bug' that is related more to how windows works. Read more on that one here: http://forums.sandboxie.com/phpBB3/view ... 11&t=21319
If it's the first, remove the name from the forced program list. If it's the second, a reboot after cutting and pasting the folder that was previously forced (or a sub directory of a forced folder) should fix it and allow you to run it outside of sandboxie.
