Re: How to add programs permanently

Posted: Wed Apr 23, 2014 3:10 pm
by Guest10
Maybe it's there, but you need to use the "Run From Start Menu" for that sandbox.
If the program added a sandboxed shortcut to the Desktop or the Start Menu when you installed it, then you should be able to find that shortcut using the "Run From Start Menu" option for the sandbox.

Either right-click the Sandboxie Control tray icon and hover the mouse above the sandbox name, or right-click the sandbox in Sandboxie Control's window, to find that Menu.

If the program did not add a sandboxed Desktop or Start Menu shortcut when it was installed in the sandbox, you need to create a shortcut to its .exe program in the sandbox.
