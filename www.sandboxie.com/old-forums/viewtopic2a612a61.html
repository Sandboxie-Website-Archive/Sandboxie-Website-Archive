josh358, I am not sure what your issue is but this are possible solutions: 1. Delete the sandbox if you dont have it done automatically when Firefox closes, 2. Create a new sandbox and run Firefox in it. Perhaps a setting in the sandbox where you are running Firefox is messed up, 3. Disable Flash Protected mode, 4. Your antivirus. Some antiviruses cause delays when opening and closing sandboxes. I dont use an antivirus but when I did, I remember some really caused long delays. Good luck.

Bo

This solution for Really speeding up... IE11/ Firefox... loading..., with my ESET Smart Security 7 AV (may work with other AV's too?)-
1. Put Firewall in Learning Mode (Do NOT browse, as FW protection is Disabled!)/reboot.
2. Open/close IE11/FF (permanent rules are created!). Now open/close them again for new AV rules to be Trusted...
3. Open/close IE11/FF in sandboxie (more permanent rules). Now open/close them again for AV rules to be Trusted...

4. Now open/close your other applications (Twice!), that use the internet- CCleaner, Macrium Reflect, DVD copy, Anti-logger free, ...
4a. Put your FW back in Automatic/Interactive Mode/Reboot, and you are good to go.
