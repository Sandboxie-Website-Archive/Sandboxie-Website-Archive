First thing to try, after making the changes, delete the sandbox if the sandbox is not set to delete on closing the browser. Then try running the browser in the sandbox.

If what I wrote above is not the reason for your problem and the solution, after you made the changes and deleted the sandbox, Multiprocess Windows still displays as 0/1 (Disabled) in Firefox Help>Troubleshooting Information, then perhaps your Firefox profile is corrupted. If this is the case, I would create a new profile.

Code: Select all

OpenFilePath=firefox.exe,%UserProfile%\AppData\LocalLow\Mozilla\temp-*
For example:

AutoDelete=y

BorderColor=#00FFFF
