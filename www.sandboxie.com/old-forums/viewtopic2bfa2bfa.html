Post by tzuk » Sat Dec 13, 2008 7:55 am

The EditAdminOnly is about preventing non-administrators from configuring some parts of Sandboxie. There is no feature in Sandboxie at this time that can prevent particular users from using its functionality.

Having said that, you could create a local or domain user group, let's say it's called SbieDeniedUsers. Then view properties for the folder C:\Program Files\Sandboxie and go to the Security tab. Click Add and enter SbieDeniedUsers. Then for this group, mark the Deny column for the Full Control access. Repeat for the .EXE files and .DLL within that folder. Finally, associate some users with SbieDeniedUsers.
tzuk
