Hello north450,

I downloaded Vivaldi and Torch and was able to launch them Sandoxed without issues. Were they working before you updated Windows?
Are you installing them inside Sandboxie?
Did you try a new sandbox with default settings?

Please try disabling your Antivirus software and see if that makes a difference.

If the problem persists, please paste the output of your Resource Access Monitor, using the "</>" button to format it.
https://www.sandboxie.com/index.php?Res ... essMonitor

Regarding installing Firefox 55 inside the Sandbox:
viewtopic.php?f=11&t=21974&start=75#p129610

If you tried to install Torch inside Sandoxie, you will need to un-tick the open videos and the other option listed at the end of the installation, and then to run it, you will need to deactivate its Sandbox, just like you would do if you were installing Chrome inside Sandoxie

Navigate to the installation folder inside the Sandbox, right click on Torch.exe --> Create Shortcut
Drag the shortcut to a Sandboxed Desktop, recover it
Right click on the shortcut and add the following at the end of the Target section
--no-sandbox
Apply/Ok.
Launch Torch via the modified shortcut.

Regarding installing Vivaldi in the Sandbox, it hasn't been tested in a while. I can see it crashes when trying to install it.
viewtopic.php?f=11&t=21974&p=115010&hil ... di#p115010
If you unzip the installer, then unzip it again, there is a Vivaldi.exe that you can run Sandboxed without issues (so far).
