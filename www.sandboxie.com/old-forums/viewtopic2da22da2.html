 How can I save my existing settings to use when I build a new sandbox?
Windy, for testing the new sandbox, you ll probably want to test using the new sandbox with default settings. If some setting is messing things up in the old sandbox for you, and you copy settings when you create the new sandbox or reinstall SBIE saving settings, then you ll probably end up with same glitch. So, you want to test the new sandbox with default settings, to see if recovery works as is supposed to.
