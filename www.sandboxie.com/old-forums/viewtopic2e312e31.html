Awesome software!

I'm using Sandboxie for years. Since I'm using this program, I've got no more problems with virusses or keyloggers or other kind of malware.

For example, it's not possible to run Teamviewer without lowering the security settings.

The program is also ideal for test installations. Only programs, that are needing deep access to the system won't work correctly.

I recommend everyone the commercial version of this fantastic program. Not only to to support the developers. It's possible to run serveral sandboxes at once. As a webmaster, I've installed Firefox and opera in a special sandbox, cause I don't need the programs on my system.
