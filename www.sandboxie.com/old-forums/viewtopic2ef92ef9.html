Combined 32/64 installer:
http://www.sandboxie.com/SandboxieInstall.exe

Separate:
http://www.sandboxie.com/SandboxieInstall32.exe
http://www.sandboxie.com/SandboxieInstall64.exe

Changes in 5.14 RC
1) Added OpenClsId={3480A401-BDE9-4407-BC02-798A866AC051} under Thunderbird in templates.ini.

Changes in 5.13.8
1) Fixed hang when starting MS Office applications.
2) Eliminated error SBIE2314 Cancelling process xxxx.exe [xxxx / 9]
3) Added Vivaldi settings to templates.ini

Changes in 5.13.7
1) Proxy Auto Config (PAC) files were not functioning in Win 10 Anniversary.

Changes in 5.13.6
1) Fixed issue some users had with blank web pages in Chrome 53.

Changes in 5.13.5
1) Fixed Recover to Any Folder browse dialog box.

Changes in 5.13.4
1) Vivaldi is now supported.
2) All browse for folder dialog boxes have been updated. They are now larger, re-sizable and have more features.

Changes in 5.13.3
1) Added new Sbie setting BlockNetworkFiles. It is available under Sandboxie Settings -> Restrictions.
When BlockNetworkFiles=y, sandboxed applications are blocked from reading network files or folders. Individual files/folders can be opened for reading/writing using the normal Resource Access settings.
This option is enabled by default for new sandboxes.
2) Eliminated the SBIE2314 Canceling process SandboxieBITS.exe [xxxx / 9]
3) Fixed the Microsoft Account Sign-in Assistant issues (hotmail.com, outlook.com, etc.)


Changes in 5.13.2
1) Fixed clipboard copy/paste failure from sandbox to host.

Changes in 5.13.1
1) Fixed Actual Window Manager (and maybe some other apps).
