bawldiggle, if you are browsing running PM or Firefox sandboxed and you click on a PDF link, your PDf reader runs the PDF sandboxed.

In the free version you can not use multiple sandboxes at the same time. Using more than one sandbox at a time is one of the features that make Sandboxie a great program. But the free version is fantastic as it is and using it gives you a good taste of what SBIE is about. I suggest you test what Sandboxies free version has to offer and in a short while, you ll know if getting the license is worth it for you.

You can copy from a sandboxed browser and paste it to a (for example) text file. I copy to the clipboard all the time while running my PDF reader sandboxed..

Most programs install in a sandbox but not all. I find SBIE great for trialing things like browsers, addons, Java, flash, PDF readers, video players, other plugins, etc. If you install something in a sandbox and you like the program, after deleting the sandbox you should install it in your real system. Sandboxie doesnt offer a "move" kind of option. Even if it did, I probably wouldn't use it.

You found a great program, I hope you like it as I do. Sandboxie is the most important program in my computer and the only one that I cant replace. When I am using my computers, I am always using SBIE. No matter what kind of program I am running, the program runs sandboxed.

You are welcome, bawldiggle.

You get same protection running the browser sandboxed in the free as in the paid version. If you click on a PDF link while running sandboxed, the PDF runs in the sandbox where your sandboxed browser is running.

If you use the PDF plugin, the PDF displays within the browser. Or you can do like me. I don't install the plugin and when I click on a PDF link, the PDF runs sandboxed out of the browser.

The paid version makes things automatic, easier to get programs and files running sandboxed, less thinking required. More comfortable getting programs running sandboxed. And better isolation. By using multiple sandboxes, you separate programs from each other. Thats what the paid version gives you. And its plenty.
