Hidden Benefit

Posted: Tue Jan 06, 2009 9:16 am
by PanamaVet
I became a registered user about a month ago and really appreciate the work tzuk has done.

I like to have two hard drives in my system so I can optimize the data throughput. I put items like the Windows temp directory and temporary browser files on the non-system drive.

Using FileRootPath=D:\Sandbox\DefaultBox in the Sandbox configuration I can move all the disk writes in a sandboxed environment to the non-system drive. Application response is incredibly fast.

Thanks again, tzuk.
