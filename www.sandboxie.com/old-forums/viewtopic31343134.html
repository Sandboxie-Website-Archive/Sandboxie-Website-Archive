Post by Guest10 » Sat Aug 19, 2017 9:35 am

I assume that you mean the "Run Sandboxed" entry that appears on the File Manager (Windows Explorer) right-click menu.
A reinstall should put two "Run Sandboxed" entries in the Registry. One is for use with files, and the other is used with folders.
You could look to see if they are there, using the Registry Editor, although I have only listed the one that's used with files, below.

Run > regedit <ENTER>

Navigate in the left window pane to:
HKEY_CLASSES_ROOT\*\shell\sandbox
Regedit pic.jpg (41.65 KiB) Viewed 742 times
You should see in the right pane:

Code: Select all

(Default)      REG_SZ      Run &Sandboxed
Icon           REG_SZ      "C:\Program Files\Sandboxie\Start.exe"
In the left pane, right underneath "sandbox", click on: command
In the right pane you should see:

Code: Select all

(Default)      REG_SZ      "C:\Program Files\Sandboxie\Start.exe" /box:_ask_ "%1" %*
Are they there?

For completeness, see if the other Sandboxie "Run Sandboxed" entry is also there:

HKEY_CLASSES_ROOT\Folder\shell\sandbox

Code: Select all

(Default)      REG_SZ      Run &Sandboxed
Icon           REG_SZ      "C:\Program Files\Sandboxie\Start.exe"
HKEY_CLASSES_ROOT\Folder\shell\sandbox\command

Code: Select all

(Default)      REG_SZ    "C:\Program Files\Sandboxie\Start.exe" /box:__ask__ "C:\WINDOWS\explorer.exe" "%1"
Maybe, check to see if that one is there. It should be put there by a Sandboxie install.
