My use case for Sandboxie isn't to protect myself from malware but rather to make it so I can easily install programs, transfer the whole installation package (including configuration) to another computer, and remove the application when done without it leaving traces all over my system. Is it possible to disable the other features and only proxy file IO and registry IO through Sandboxie?
