Post by Craig@Invincea » Thu Aug 04, 2016 12:06 pm

When you upgraded to Windows 10, MS made Edge your default browser. Edge does not work with SBIE.

You need to set your default browser in Windows to something other than Edge. You then need to set tbird to use a browser of your choice, otherwise, it will use the default.

This is typical Windows behavior and not related to SBIE.

Post by RT2 » Thu Aug 04, 2016 12:43 pm

After more investigations, I found the solution to my problem. When I tried to create a new sandbox, I didn't pay enough attention and copied the settings of another one (my bad). With a fresh new sandbox, it worked fine. Then I went to the configuration file and played with every settings, and I found that adding "Template=qWave" to the sandbox I use for browsing with Google Chrome solved the crashes.

My sandbox was just outdated, and now I know it is better to check the settings of a default sandbox from time to time in order to add the new ones to my old sandboxes.
