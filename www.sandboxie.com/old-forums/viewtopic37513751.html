Hello! Is it possible to migrate a program from a sandbox to a real environment? Thanks in advance!

Posted: Tue Sep 11, 2012 4:00 am
by oloid
No, basically. There is no automated way to export an installed program and all components in it's entirety. But you could try copying a program folder to your real system, whether it would run normally or not would depend on it's installation complexity. Some programs rebuild their necessary registry entries or settings when run, others would just fail spectacularly due to missing bits, which might include files installed outside the program's folder.
