Antivirus/Anti-malware|SBIE v5+[working/not][Updated 7/14/17]

  • Quote

Post by Craig@Invincea » Tue Aug 04, 2015 3:06 pm

Compatible or non-compatible Antivirus / Anti-malware [and some firewalls] when running SBIE v5+.
We recommend Windows Defender / Essentials as your 2nd layer antivirus/anti-malware.
Remember, if your SB becomes infected, simply delete your sandbox contents to remove the virus & malware.
[ SBIE Virus FAQ http://www.sandboxie.com/index.php?FAQ_Virus ]
Tested w/ SBIE 5+ and this thread will also apply to Windows 7, 8.1 and 10 [concentration on Win 10][Standard install,standard options/scanning & browser extensions]
List will be updated as we test and with user submissions.

NOT Currently Supported
Kaspersky Internet 2015/16 (Windows 10, and 8/8.1 as well) (Must be removed for SBIE to function correctly) [On Win 7/8.1 that have SBIE excluded in KIS 2015, users have had some success ]
Some Zemana on demand products may cause errors.
Norton Security 2015/15 [See below for related Norton/Symantec products]
Norton 360 [viewtopic.php?p=130045#p130045]
Symantec Endpoint Protection v14 [viewtopic.php?p=130045#p130045]
Dr.Web Katana
*HitManPro may cause this error when using Chrome:
SBIE2203 Failed to communicate with Sandboxie Service: *GUIPROXY_0000000x - chrome.exe errors, if so, disabling HMP solves the issue. http://forums.sandboxie.com/phpBB3/view ... an#p118267 [If Chrome appears to be working, you can just hide the message, but it will continue when SBIE starts.]

Supported AV related [Not an endorsement][YMMV]
WebRoot Antivirus
Hitman Pro Alert Trial, v3.1.7 Build 357 [Tested on Windows 10x64]
ESET Smart Security 8,9 [Windows 10]
ESET NOD32 ANTIVIRUS 9
Comodo [You must do this to enable it to work - no guarantee.. http://forums.sandboxie.com/phpBB3/view ... o&start=30 ] ALSO try this for Win 10 in this thread http://forums.sandboxie.com/phpBB3/view ... 80#p120580. If printing fails, add splwow64.exe to the list of excluded processes.
Panda Antivirus Pro 2016 [with tool bar add-on & all options enabled]
Panda Free Antivirus 2016
AVG AntiVirus Free 2015(w or w/o ZEN) [Supported in Beta 5.21.4 viewtopic.php?p=129389#p129389 ----> If needed, see viewtopic.php?f=11&t=24536&start=15#p128578 for workarounds]
AVG Security Suite w/ Surf Shield off.. In AVG c:\program files\sandboxie to the exclusion list.
--- Under Advanced Settings->Exceptions, add folder \program files\sandboxie to the list of exclusions.
MS Security Essentials & Defender [Recommended for use with SBIE]
Norton Security Suite [As provided by Comcast / XFinity, based on Norton 360 Premier Edition minus online storage] [http://my.xfinity.com/learn/internet/security/ ]
Avast Free Antivirus v12.1.2272[build 12.1.3076.6]
BitDefender Anti Virus [Free Edition up until version 1.0.10.12 ] [Win10/64] [Supported in Beta 5.21.4 viewtopic.php?p=129389#p129389 ----> If needed, see viewtopic.php?f=11&t=21539&start=165#p128854 for workarounds]
MalwareBytes Anti-malware [v2.1.8.1057]
Avira 2016 [free version tested]
Lavasofts AdAware AV
Emsisoft Anti Malware
GData Total Protection
PC Matic [PC Pitstop]
Symantec Endpoint Protection [Mainly Corp / Enterprise Users]
Qiho 360 Internet Protection [8.2.0.1066]
KIS 2017 - [See viewtopic.php?f=11&t=21539&start=165#p125871 for workarounds]
Bitdefender (Total Security, Internet Security 2016-2018) [SBIE Error2224 SBIEDComLaunch.exe has crashed][Win10] [Supported in Beta 5.21.4 viewtopic.php?p=129389#p129389 ---- > If needed, see viewtopic.php?f=11&t=21539&start=180#p129505]

Unsupported Firewalls
Zone Alarm
Comodo Firewall [reports indicate it may not work w/ SBIE]
