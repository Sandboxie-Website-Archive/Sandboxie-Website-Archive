  • Quote

Post by incognito555 » Wed Jun 19, 2013 3:39 pm

I'm experiences a couple of errors with the new SB 4.02 and StrokeIT used on a sandboxied Chrome v27, so I'll start with the first error which just generates a warning message BUT doesn't effect usage, also found here: http://www.sandboxie.com/phpbb/viewtopic.php?p=90014

SBIE2203 Failed to communicate with Sandboxie Service: *GUIPROXY_00000001 - chrome.exe [C0000001]

How to replicate (win7 64bit and test in VM win7 32bit)

1. Install Stroke It http://download.tcbmi.com/ZQ429IO/strok ... nglish.exe
2. Enable detected StrokeIT software compatibility template in Sandboxie
3. Launched a sandboxed chrome v27
4. Test to make sure you can draw strokeit line gestures in chrome
4. In the URL or any text field, or anywhere perhaps, hold the right mouse button and scroll the mousewheel up and down a few times
