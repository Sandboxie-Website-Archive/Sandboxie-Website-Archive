I figured out the problem. Apparently sandboxie had tried to update chrome inside the box, which generally doesn't work and causes issues. So I had run chrome outside the box which caused a profile error and created some silly process that for some reason didn't allow chrome to start inside the sandbox. Long story short I removed the processes once I saw them and now all appears to be good!
