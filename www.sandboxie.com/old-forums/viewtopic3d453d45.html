Hi Curt,

I'm using the windows right click context menu "copy" and "paste."

[Beta 4.17.8]

Steps to reproduce issue (Windows 7 SP1 x86) when copying files from outside the sandbox and pasting into a sandbox folder:
  • 1. (Non-sandbox Desktop) Right click-->New-->Text Document (name it "Sandbox.txt")
    2. Open the "Sandbox.txt" file and type in "Testing" and save the file
    3. Click on Sandboxie Control and click on "Sandbox DefaultBox"
    4. Right click-->Run Sandboxed-->Run Windows Explorer (Sandbox Windows Explorer)
    5. (Sandbox Windows Explorer) - go to any sandbox folder (ex. Sandbox "My Documents" - C:\Users\User\Documents)
    6. (Non-sandbox Desktop) Right click on "Sandbox.txt" and pick "copy".
    7. Right click on a blank area within the (Sandbox "My Documents" folder) and pick "paste"
    8. Observe that the "Sandbox.txt" file does not appear in the (Sandbox "My Documents" folder)
Steps to reproduce issue (Windows 7 SP1 x86) when copying files from one sandbox folder and pasting to another sandbox folder:
  • 1. Click on Sandboxie Control and click on "Sandbox DefaultBox"
    2. Right click-->Run Sandboxed-->Run Windows Explorer (Sandbox Windows Explorer)
    3. (Sandbox Windows Explorer) - go to any sandbox folder (ex. Sandbox "My Documents" - C:\Users\User\Documents)
    4. Right click on a blank area within the (Sandbox "My Documents" folder) and pick "New"-->Text Document (name it "Digital.txt")
    5. Open "Digital.txt" and type in "Testing" and save the file
    6. Right click on "Digital.txt" and choose "copy"
    7. (Sandbox "My Documents" folder) - go to another sandbox folder such as (ex. Sandbox "Downloads" - C:\Users\User\Downloads)
    8. Right click on a blank area within the (Sandbox "Downloads" folder) and pick "paste"
    9. Observe that the "Digital.txt" file does not appear in the (Sandbox "Downloads" folder)
No issue when copying a file from a sandbox folder to outside the sandbox:
  • 1. Create "Virtual.txt" file in (Sandbox "Downloads" folder)
    2. Open "Virtual.txt" file and type in "Testing" and save it
    3. Right click on "Virtual.txt" and choose "copy"
    4. Go to (non-sandbox Desktop) and right click-->"paste" ("Virtual.txt" file appears in non-sandbox Desktop)
As for another copying method, I added this registry key a couple of years ago (http://www.howtogeek.com/howto/windows/ ... lick-menu/) outside the sandbox. This key adds "copy to folder" to the right click context menu. "Copy to folder" does work when copying files from one sandbox folder to another sandbox folder.

Command Prompt also works if I use the copy command (Run Sandboxed-->Run Any Program-->Type in cmd-->Type in command: copy C:\Users\User\Downloads\Virtual.txt C:\Users\User\Documents)
