Sandbox Observer

After starting Sandbox Observer it continuously checks all sandboxes for new processes and shows the name of the sandbox and process in a balloon tip.




Features

- File verification based on previously saved hash
- Exclusion list for sandboxes and processes
- Check sandboxes for missing 'DropAdminRights' setting


Download disabled - Due to design problems this project is stopped for an indefinite time.
