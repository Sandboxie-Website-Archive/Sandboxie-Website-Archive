About the PDF that is attached to an email, if you are using web mail, as long as the browser is running sandboxed, the PDF or any other attachment that you run, will run sandboxed.

You can also force your PDF reader to make it run sandboxed automatically every time a PDF runs by using the Forced programs feature that gets unlocked in Sandboxies paid version. Read the link posted by Barb.
