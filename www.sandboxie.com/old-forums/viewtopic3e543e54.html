swhtmw wrote:Can you plan Sandboxie, that i can do DefaultBox Sandboxie Conainer on Z:\ and Sandboxie Box 2 - Container on C:\ ?
You can already do this using FileRootPath.
To do this, just open Sandboxie Control, click Configure -> Edit Configuration and scroll down til you see your second sandbox listed, such as [Sandbox2].
Below that enter a FileRootPath location, in your case it would be something like FileRootPath=C:\%SANDBOX%, then save the file.

swhtmw wrote:For users with more than one sandbox: Can you creat the menue with right click on a programm with all sandboxes
This sounds like what we already have?
swhtmw wrote:Maybe I will buy the full version of sandboxie. Than I can creat more than one Sandbox. But my DefaultBox is on Z:\ RamDisk (all content delete after closed all programms).
If I creat "Sandbox 2" I would to have the Sandbox 2 Container not on RamDisk. The Sandbox (Sandbox 2 - Container) must be on Harddisk (C:\) an don't delete after closed all programms in Sandbox 2.
All versions of Sandboxie can create new sandboxes. You can create as many as you want, but you need a registered version of Sandboxie to use more than one of them at a time.
Add the line shown below, using Configure > Edit Configuration, after you create the new sandbox "Sandbox_2".

[Sandbox_2]

FileRootPath=C:\Sandbox\%SANDBOX%
...
...
...
swhtmw wrote:If I buy the full version, can I use the full version on my pc AND on my laptop?
Yes.

[Edit] You can also use Sandboxie Control to create a Desktop shortcut that will run a program using the sandbox you select.
Use the menu: Configure > Windows Shell Integration
and look for the button "Add Shortcut Icons".
http://www.sandboxie.com/index.php?ConfigureMenu#shell
Select the sandbox to use and then select the program to run sandboxed, from the Sandboxie Start Menu that opens.
