by Curt@invincea
Bo, we're trying to add more detail to the SBIE2314 to find out why some processes are terminating, so we added [PID / reason code]. Reason code = 0 means that the process never really started. Most likely due to a DLL being blocked. So for reason code = 0, I will change it so the PID and reason are not part of the error message, so hiding it will work as before. Thanks for letting us know. Are you getting any other reason codes besides 0?
