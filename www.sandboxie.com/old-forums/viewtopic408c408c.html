by Barb@Invincea
Hello Numbat,

The Windows SmartScreen template was introduced with Sandboxie beta 5.21.7. If you manually added it to your sandboxes, you will have to remove it from each one of them before/after installing Sandboxie 5.22 (otherwise, you will receive an error message stating that the template doesn't exist).
You can follow the steps posted in the beta notes in order to remove it:
(...) Created a template to address "SBIE2101" issues caused by Windows Defender "App & Browser Control" in Windows 8.1 and 10. This needs to be added manually by right-clicking on your Sandbox ---> Sandbox Settings --> Applications ---> Miscellaneous ---> Windows SmartScreen [The template will be added automatically in future versions of Sandboxie]...
You can also go to Configure--> Edit Configuration
Search for any Windows SmartScreen references and delete them.
Save the file, close it.
Configure--> Reload configuration.

Starting with release version 5.22, Sanboxie handles this at the code level, so there is no need for a template.

Regards,
Barb.-
