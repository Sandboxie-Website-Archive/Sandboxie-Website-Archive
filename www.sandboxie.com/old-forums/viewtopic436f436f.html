Sandboxie version 1.15 Beta Released

Posted: Fri Aug 20, 2004 4:30 pm
by tzuk
This is a bug fix release.

- It should now install on Windows XP Service Pack 1a.

- It should now work well on a FAT filesystem.

- If it crashed your box when you asked to Terminate All Sandboxed Processes, it shouldn't do that anymore.

- The Sandboxie Control main window should not become inaccessible anymore.

- Sandboxie Control will display a nice message when you ask to Delete Contents of Sandbox and some files are inaccessible. This replaces the cryptic message about system code C0000043.
