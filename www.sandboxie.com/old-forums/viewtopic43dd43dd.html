Post by JoeHood » Sun Apr 16, 2017 10:24 am

This seems to be an Application Path issue in the Windows registry. If you have ForceProcess=firefoxportable.exe already set and then relocate the portable folder, you get an error.

If you run it unsandboxed, it seems the app resets the registry to the new file path. But if you change the location and then run it sandboxed you get the error.
