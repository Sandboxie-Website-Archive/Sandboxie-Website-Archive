0xc00000e5 = STATUS_INTERNAL_ERROR. Which is obviously not one of the most helpful error msgs (Sbie does not return this error anywhere).

I would suspect that the error is coming from some AV conflict. Have you tried removing Comodo?

Also, Craig is right about the Intel hardware virtualization stuff. Everyone at Invincea has that enabled because it is required for VMWare which we use for testing. It does not affect Sbie.
