Hi,
I believe sandboxie uses the system code to identify whether the installation is on a new PC. However, I find the system code changes quite frequently even when it is the exact same PC. For example, it can change after some windows update, or after reinstalling windows, or after installing new graphics card driver or printer driver, or after installing some software, or even just by updating sandboxie.

I did some experiments and have confirmed this behavior: Same PC,windows 10 digital license (activation without a key), same sandboxie installer, got different system code depending on the SW / drivers in the system.

This is very frustrating, and already causing my licences being cancelled due to "activated on too many computers". This seriously feels like a time bomb. Do you have any suggestion to avoid this? or and plans to improve it?
