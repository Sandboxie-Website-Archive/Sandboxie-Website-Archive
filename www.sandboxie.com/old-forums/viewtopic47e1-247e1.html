Im noticing that Chrome Version 28.0.1500.72m does not display Rich text notifications while under sandbox with Checker plus for G mail 13.6.1 Extension. As you might know, Chrome has implemented a feature where chrome extensions may utilize an API to display Rich text Notifications ( a flyout) in task bar such as when new emails arrive.

Out of Sandbox guard it displays fine. Currently on Windows 7 64 bit using Chrome Version mentioned above with latest Sandboxie 4.05.02.

The easiest way to check that these notifications work is to open up Chromes task manager under tools, select an extension and End Process. This should produce a rich text notification saying the extension has crashed on bottom right of screen.

On a side issue, im also seeing performance slowdown of Chrome running in its own sandbox on the odd occasion and am unable to pinpoint the cause. Sometimes when PC resumes from sleep is see double sandboxie icons or a double X symbol of Sandboxie trying to terminate a process even though, before PC was put to sleep all programs were terminated. This only occurs on the odd occasion and am unable to reproduce.
