Hi Tzuk, running Libre portable in SBIE 4.01.05 is better now (W7 32bits laptop). Before .05, I needed a workaround in order to make forcing Calc, Impress and Writer work. I had to force 7 executables and allow Start/Run access to 8 executables in order for those programs to run forced in a restricted sandbox. The only remaining issue that I can tell between SBIE and Libre is that I am still getting a 2205 message when closing Writer. Before beta .05, I was getting a 2205 message when I opened and closed Writter, Impress or Calc.
