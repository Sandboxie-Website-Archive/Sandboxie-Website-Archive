Sandboxie.ini can also be located in the Sandboxie program directory.
Ex: C:\Program Files\Sandboxie\sandboxie.ini

If you move it there, then use Sandboxie's
Configure | Reload Configuration
right away, so Sandboxie will find it.

by tzuk
Sandboxie frequently rewrites the Sandboxie.ini file, so it would be impractical to just offer an Include directive/instruction, without a drastic change elsewhere that knows to differentiate which settings come from the "primary" file, and which from the "included" file. And such a change would not be trivial.

Thus I would suggest the following "workaround". Install Sandboxie to the folder that is shared between the systems. Then follow Guest10's suggestion to move Sandboxie.ini to that folder and reload configuration.
