Post by Guest10 » Wed Nov 13, 2013 11:27 am

If "Internet Download Manager" shows with a [+] at:
Sandboxie Control (tray icon) window > Configure > Software Compatibility, then select it and click on the "Remove" button > OK.

That item will then show as [ ], and it will change the entry under Configure > Edit Configuration > [GlobalSettings]
from:
Template=Internet Download Manager
to:
TemplateReject=Internet Download Manager

I don't use IDM, so I can only go by the way my old version of Orbit downloader works with its template marked as "TemplateReject":
The IDM template will no longer be used. If IDM is started by a sandboxed program, then another instance of IDM should start (sandboxed) and save the download inside of the sandbox.
