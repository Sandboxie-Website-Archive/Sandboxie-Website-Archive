broughie wrote:craig i have read that Sandboxie can prevent exploits damaging the pc system ,so is MBAE really necessary to run within it ?
SBIE doesn't prevent/block malware from "running" in the sandbox if it get's downloaded. It only prevents it from making changes to your machine/software and accessing anything outside of that sandboxed container.

If you're curious what may be living in a sandbox, It helps to run Anti-malware and items like MBAE to "look" at your sandbox and know what could be there. If you find something, no need to have those programs do anything, simply delete the contents of your sandbox. It's more of...What have I caught today sort of info. Happily knowing you're okay.

There is a MBAE template in the Contributed Utilities thread.

shmu26 wrote:So I just need to paste the MBAE template at the bottom of Sandboxie.ini , and save?
No.
When nothing is running sandboxed...
Use Sandboxie Control's menu: Configure > Edit Configuration.
Scroll to the bottom of the file; add one blank line, to separate existing settings from the new template; and then Paste the template at the bottom of the file.
Then click File > Save, File > Exit.

If you have only one sandbox where you will use the template, then: Sandbox Settings > Applications > Security/Privacy.
Click on the line "[ ] Malwarebytes Anti-Exploit", and then click Add, and then OK.
This is the approved method of Adding a template to the settings for a sandbox.

If you have other sandboxes (but not all sandboxes) in which you want to use that template, then use the same procedure for each of them.

If you want the template to apply to all of your current and future sandboxes then you can to make it a global setting:
After Add(ing) the template to one sandbox using the above procedure, open Sandboxie Control's menu: Configure > Edit Configuration.
Scroll down to the sandbox where you Add(ed) the template and you will see the line:
Template=MBAE
in the settings for that sandbox.

Place a copy of that line underneath any existing settings that you may have underneath the Global Settings section, or just type the line in yourself - and make sure that there's a blank line after that line.


[GlobalSettings]

...
...
... <-- Your current settings, if any
...
Template=MBAE
<-- one blank line should follow the above line. Especially if there's another [........] section below it.

Then Save, Exit.
----
I said, above, to Copy the line - but really, you can Cut that line since you won't need it in the settings for the sandbox where you Add(ed) it. Just be sure to delete any blank line that results when you Cut the line. You don't want any blank lines within the settings for a sandbox.
Also, as an alternative method: After Pasting the template into your configuration file, you could just skip most of the above and just type or paste Template=MBAE into the settings for any or all sandboxes, or under the [GlobalSettings] section. Just be sure to allow for a blank line in between it and any other [......] section that follows it.
As long as you used Configure > Edit Configuration, then Control will automatically Reload the configuration. If you manually edited the sandboxie.ini file in the Windows folder, you would need to use Control's Configure > Reload Configuration command to load the configuration changes into memory.
