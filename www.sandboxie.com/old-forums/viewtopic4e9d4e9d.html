Hi!
After installing KB4019215 for Windows 8.1 today I got the Sandboxie warning panel showing up repeatedly when surfing in Chrome saying that a few things weren't working right. As I feel by far more protected by SB than Windows Updates I uninstalled that update. The other updates today did not cause that problem so I didn't remove them.
