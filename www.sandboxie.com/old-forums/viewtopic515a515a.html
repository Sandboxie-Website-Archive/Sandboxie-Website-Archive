Thanks for your fast reply. But that very old post doesn't help. The template [Template_OfficeClickToRun] doesn't exist in 2009.

It doesn't make much sense to me that the installer allows changing the path, if then the Templates.ini ignores this changed path.

Please add an Expandable Variable for the installation path. Or maybe you can add the following changes to the Templates.ini:

Code: Select all

[TemplateSettings]
Tmpl.Sandboxie=%ProgramFiles%\Sandboxie

[Template_OfficeClickToRun]
HostInjectDll=%Tmpl.Sandboxie%\SboxHostDll.dll
HostInjectDll64=%Tmpl.Sandboxie%\SboxHostDll.dll
Users using a changed installation path could add the following to their Sandboxie.ini:

Code: Select all

[GlobalSettings]
Ovr.Tmpl.Sandboxie=X:\path\to\sandboxie
