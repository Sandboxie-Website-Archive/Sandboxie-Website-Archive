hoanglong2711 wrote:I want save file render complete of Camtasia Studio on my HDD. After, render complete I can't find file in my HDD. Can I setting for sandboxie enable save file direct to my HDD ?
To save files to the hard drive after they are created, you need to set Camtasia and Sandboxie to save at the same folder. For example, if you set Camtasia to save files at the Desktop, files that get created in the Desktop will be recoverable in Sandboxie because Desktop by default is included as one of the Quick recovery folders in Sandbox settings.

You can also save file to the HD that are created at the location that Camtasia uses by default by you adding that folder in Sandbox settings>Recovery>Quick recovery.

About the license. Home user should be fine. One key. And with the license, you should be able to open 4 instances of Cantasia IF Camtasia allows that to be done.

Bo

hoanglong2711 wrote:Thank you. Please guide me more. I want save to location: G:/Rendered. Help me setting it in sandboxie.
You need to make G:/Rendered a Quick recovery folder.

Sandbox settings>Recovery>Quick recovery, Click Add folder, navigate to folder G:/Rendered, Click Apply and OK.

Make sure Camtasia is also set to save files at that same folder.
