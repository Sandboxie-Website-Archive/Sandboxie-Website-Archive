Hello windows2045,

I installed AVAST 18.2.2328 (this is the current version) on Win 10 x64 + Sbie 5.23.6 and tried to launch an app in the Sandbox, it immediately started scanning it (took several seconds), after it stated that nothing wrong was found, it started yet another scan and so on..It delayed all the processes in the Sandbox until they crashed. I did not receive the message you received, however.

I added an exclusion and I was able to start launching apps without issues:
Avast --> Settings ---> Exclusions
File Paths --> Browse
Find your Sandboxie installation path (for example C:\Program Files\Sandboxie) and click on the check box next to it.
Hit OK
Hit ADD
Hit OK (green button)

Re-try launching apps in Sandboxie (if needed, delete the contents of your Sandbox or create a new one).

Do you have any customized settings in Avast? Any proxies, anything like that?
Did you try the latest beta / using a new Sandbox with default settings?
Is the Avast template active ? (Configure --> Software compatibility ---> "avast! Antivirus" should be listed and checked.

If all of the above listed fails, try fully removing Avast, then downloading the latest version and re-try using it along with the latest version of sbie.
