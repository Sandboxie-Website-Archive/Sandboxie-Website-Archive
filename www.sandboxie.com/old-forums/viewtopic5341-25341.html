Because if MS patches Windows, you need to patch Windows. Same for a browser. SBIE isolates your online space but nothing is 100%, that's impossible unless you totally disconnect.

Compatibility / Stability / Kernel Security / Etc / Etc.

You cannot run everything in a Sandbox. That's not possible. Relying on 1 option is a bad option. Same goes for backing up. A single backup is not a backup.

Even the author/creator states it's not 100% http://www.sandboxie.com/index.php?FAQ_Virus

There are still threats, compatibility, etc. that isolation may not be fool proof. And SBIE is only as good as the end user using it.
