[.09] Removed Low-Level and Hardware Access Settings

Posted: Sat May 25, 2013 11:01 am
by tzuk
I removed the Low-Level Access and Hardware Access settings pages from the Sandbox Settings window in version 4.01.09.
Reminder screenshot:

http://www.sandboxie.com/index.php?Rest ... s#lowlevel
http://www.sandboxie.com/index.php?Rest ... s#hardware

With version 4, many of these settings do not function correctly anymore, that is to say, they no longer make it possible for a program in the sandbox to do stuff outside the sandbox.

I feel these were rather esoteric settings that saw little use, so I don't t think they justify the time investment to make them work in version 4. Some of them are even incompatible with the way things work under the hood in version 4.

I would point out that I feel the setting "Permit programs in this sandbox to simulate keyboard and mouse input" was perhaps used mostly to get rid of message SBIE1304 Blocked simulated keyboard or mouse input by process program.exe. So in version 4.01.09 Sandboxie no longer issues message SBIE1304.

Please note that behind the scenes Sandboxie still recognizes at this time some of these settings, if they appear in Sandboxie.ini. This may change in the future.
