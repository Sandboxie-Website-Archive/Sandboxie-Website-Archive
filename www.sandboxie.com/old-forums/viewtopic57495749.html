SBIE Control context menu suggestion

Posted: Wed Aug 23, 2017 9:08 pm
by kyles
Current SBIE Control context menu, the Run series items was cascaded to one "Run Sandboxed" sub-menu.
For example, if we want to use "Run From Start Menu".
We need the following steps:
1. Right click on Sandbox name.
2. Move the cursor to "Run Sandboxed".
3. Wait around 0.5sec for the sub-menu appears.
4. Move the cursor horizontally (within the "Run Sandboxed" height) to the right.
5. Move the cursor down to "Run From Start Menu"
6. Click
