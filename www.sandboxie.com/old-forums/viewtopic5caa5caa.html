Post by SnDPhoenix » Wed Jul 18, 2007 2:30 pm

Ok, lets do a trace then, go to your configuration and look for this:

FileTrace=.
PipeTrace=.
KeyTrace=.
IpcTrace=.
GuiTrace=.

change it so it looks like this:

FileTrace=.
PipeTrace=d
KeyTrace=.
IpcTrace=d
GuiTrace=d

then hit Ctrl+S on your keyboard and close the config file, then reload the configuration.

Now grab Debugview here http://www.microsoft.com/technet/sysint ... gview.mspx, once its downloaded, extract the zip file and run Debugview, once its running, go ahead and launch your game sandboxed (make sure the sandboxed game is the only instance running) and you should start to see alot of text appear in debugview, go ahead and save all the text to a log file (File > Save as in Debugview) then (if its not too long) copy and paste the contents of the log file onto here.
