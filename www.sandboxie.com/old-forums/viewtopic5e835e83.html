HitmanPro.Alert CryptoGuard 2.5.6.65 is an old version. HitmanPro.Alert 2.6.3 build 74 is the latest version. You could download that version (http://www.surfright.nl/en/alert) or wait for soon to be released HitmanPro.Alert version 3 (http://www.surfright.nl/en/home/press/s ... es-alert-3).

Datasheet version 3: http://dl.surfright.nl/Alert-3/HitmanPr ... asheet.pdf

Guest10 solution for all sandboxes: "You could do a manual edit using Configure > Edit Configuration, and move the setting so that's it's underneath the [GlobalSettings] section. That way it should apply to all sandboxes- both the existing sandboxes and all new sandboxes that you may create in the future. You'll have to decide if you really want that.

I assume that the line to move says: OpenPipePath=\Device\NamedPipe\hmpalert"
