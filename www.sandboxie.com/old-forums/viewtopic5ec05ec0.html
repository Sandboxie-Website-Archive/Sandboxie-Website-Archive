Hello JASON789,

Sandboxie's default browser is Windows' default browser. So, for example, if you set Firefox to be the default Windows browser, when you click on the Sandboxie icon "Run Web browser" , it will launch Firefox in a sandbox.

You can create a shortcut so that when you click on it it launches your default browser Sandboxed. Here are the steps:
https://www.sandboxie.com/ConfigureMenu#shell

If you want Firefox (or your preferred default browser (Edge not supported) ) to start automatically in the Sandbox , that's a paid feature and you can find more info here:
https://www.sandboxie.com/ForceProcess

Regards,
Barb.-

Hello JASON789,

You can run your email program inside the Sandbox, which will cause anything invoked from it to start sandboxed. This is how you can configure it:
https://www.sandboxie.com/ApplicationsSettings#email
https://www.sandboxie.com/TestEmailConfiguration

To force a web browser (or any program) to always run inside a sandbox regardless of how are you invoking it, you will need the feature called Force Process, please see my previous response.
