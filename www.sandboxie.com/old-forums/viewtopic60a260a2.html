Post by tzuk » Sat Mar 24, 2007 7:49 am

The problem is Vista specific, and has to do with launching programs from sandboxed folders, in Vista.

Mark -- if you look carefully you'll see that the Setup.exe program is just a shell, a wrapper -- it extracts some program (which would be created in a sandboxed folder) and then runs it.

There is/was a generic problem with Sandboxie handling this. Now, I fixed about 90% of this problem -- you will see the Setup runs, but in the end when it's supposed to invoke the program itself, that part still doesn't work -- that's the missing 10%.

Anyway, the program is actually installed and you can launch it from the Start Menu, so I'm going to leave the last 10% for a later update.
