Sandboxie Hotkeys?

  • Quote

Post by SnDPhoenix » Sat May 03, 2008 5:10 am

Alright, here is my quick request. Nothing big and it wont take that long to explain, so it'll be real quick.

One of the things I've always wanted to see, is a hotkey for the "Terminate running processes" command.

Why, cause lots of times, you will have run a program which for some reason prevents you from being able to get to the menu to terminate running processes. Maybe it hides your mouse, or it makes the screen look weird, or you are getting windows popping up everywhere, or the process in question is slowing down your pc so much that it just makes accessing windows with the mouse nearly impossible.

Well this way, you just hit the hotkey and BAM, all processes terminated.

Now I know alot of people are (or aren't) thinking "Oh sure thats great, but one, you would have to use whichever hotkey tzuks decides on which might conflict with one of my existing hotkeys. Or two, thats great, but it is going to terminate all the processes in ALL sandboxes which would suck..."

Ok well then how about the ability to specify your own hotkey for the "Terminate running programs" command, in each sandbox?
Maybe in the sandbox settings window, for each sandbox, have a "Hotkeys" section that allows you to assign your own hotkeys for each Sandboxie command (Quick recovery, Delete contents, Run Sandboxed -> Web browser, etc..) in each sandbox?

Something to think about...
