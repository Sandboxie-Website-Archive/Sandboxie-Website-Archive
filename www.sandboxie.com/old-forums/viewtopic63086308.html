This worked for me using
Windows 7 Ultimate 32bit,
Sandboxie 5.22,
Firefox 57 (Quantum),
Microsoft Security Essentials Antimalware Client Version: 4.10.209.0
Engine Version: 1.1.14306.0
Antivirus definition: 1.257.459.0
Antispyware definition: 1.257.459.0
Network Inspection System Engine Version: 2.1.14202.0
Network Inspection System Definition Version: 118.1.0.0

Found these in the Mozilla forum :
http://www.ghacks.net/2016/07/22/multi-process-firefox/
https://support.mozilla.org/en-US/questions/1175046
https://support.mozilla.org/en-US/questions/1159697
So I
1. Typed about:config<enter> in the FF 57 address bar. When the
warning screen came up, press the 'I will Be Careful' button.
At the top of the about:config page is a search bar.
2. Type in browser.tabs.remote.autostart and make sure it says false, and if not toggle it to false by double clicking the item.
3. Type in browser.tabs.remote.autostart.2 and make sure it says false, and if not toggle it to false by double clicking the item.
See the attachment
Restart Firefox 57
I have not had any performance issues since I did the above actions, although it is possible that FF would work even faster with multiprocessing enabled, but apparently not on my computer.
