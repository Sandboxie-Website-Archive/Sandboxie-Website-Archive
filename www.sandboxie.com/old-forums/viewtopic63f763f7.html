Hello Stardance,

When you start the SandboxieInstaller, you get a window showing the name and version in the Title. You can install Sandboxie without internet and do an offline activation: https://www.sandboxie.com/OfflineActivation

The link provided by Bo [https://www.sandboxie.com/AllVersions] contains the versioned files, as well as their hash (which will allow you to double check you downloaded the correct version).
