Post by zyx » Wed Apr 10, 2013 11:59 am

1. Download 7stacks and some program, for example, Opera.
2. Install Opera to sandbox.
3. Create a new folder on your desktop.
4. Go to the sandbox folder where you installed Opera, and make a shortcut of opera.exe and move it to the folder you made earlier.
5. Install and run 7stacks.
6. For "Folder to use for stacks" select the folder where your Opera shortcut is.
7. Click "Create shortcut on desktop"
8. Click that shortcut 7stacks made on your desktop and select Opera.
9. You will get that error.
