Post by Syrinx » Mon Oct 24, 2016 9:56 pm

a cold boot is a boot process in which the computer starts from a powerless state.
/yay wiki?

With Windows this also (to me) means a standard boot process (eg no hiberfile.sys). The 'fast startup' option of 8/10 is similar here as it saves some parts in a file and instead of a true boot it 'loads' these elements from a file then continues to process the rest. Hybrid is the word used here [in many cases] but for your situation, it might be an issue but to my limited knowledge an actual reboot would still resolve those. Hitting shutdown may however result in such a situation. yay for complicating things MS?

Anyway, I expect a boot into safe mode would resolve your problems and allow the box to be well and truly cleared but I also understand that you want to figure out what is going on!
