Well, in any case, Oliver, if you still want to look into this, I suggest as a first step, run a trace:

Code: Select all

FileTrace=ad 
PipeTrace=ad
Without FileTrace=i which usually just clutters up the trace;
and also PipeTrace=d which could add some useful stuff.

And if possible, show the trace for only the PGP process.
