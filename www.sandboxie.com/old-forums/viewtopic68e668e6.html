I'd like to see separate options for both a browser's history and bookmarks as separate options instead of tying them both together.
There are many times I want to save a bookmark, but not save my history of the web pages that I went to.

For Firefox, my browser. Firefox saves history and bookmarks to the same file (places.sqlite). So, if you want to save bookmarks out of the sandbox, history gets saved as well.
http://kb.mozillazine.org/Places.sqlite

To get rid of history, this is what I do. Go to Firefox Tools>Options>Privacy. For History, choose "Never remember history". Or get it done using "....custom settings for history". I use the latter, I think it works best.
