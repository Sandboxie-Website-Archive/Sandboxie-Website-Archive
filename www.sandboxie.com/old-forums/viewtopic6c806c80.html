[PLEASE READ] before posting a problem report

Posted: Mon Oct 20, 2014 3:54 pm
by Curt@invincea
When reporting a problem, please include the following:

The Windows version number? (i.e. Windows 7 32bit, 8.1 x64, etc)
Sandboxie version number (SBIE v5.23.3? 5.22?)
NAME & Version/build numbers of any applications involved? (Chrome v43, FF45 etc)
Are you running antivirus/anti-malware software? (AVG, Live Essentials..etc)
What are the exact steps to reproduce the issue? (Launch app X, perform X actions, etc)
..The more information the better

If you are getting the "Messages from Sandboxie" popup dialog with error messages, please don't post a picture.
Use the "Copy Contents to Clipboard and Close Window" button and paste the errors in your post.
This is much faster and is easier for everyone to read.

To resolve your issue faster, and save everyone time, please search the forum to see if your problem has already been reported.
You can browse the forum topics, or use the search box in the top right corner of the web page.

For example, we have over a dozen threads on problems with the VC Redistributables (vcredist_x86.exe, vcredist_x64.exe)
http://forums.sandboxie.com/phpBB3/sear ... mit=Search

If a related thread already exists, but you still need additional assistance, please respond to the existing thread rather than create a new one.

Thanks
