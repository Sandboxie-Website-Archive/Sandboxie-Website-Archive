Hello Lode,

I tested on Windows 10, and the location for the bookmarks is:
%LocalAppdata%\Epic Privacy Browser\User Data\Default\Bookmarks

So you need to right-click on your Sandbox ---> Sandbox Settings
Direct Access
Edit/Add
And paste the path I provided.

If you are using a custom path, or a different OS, you will have to adjust it to match whichever location contains that file.
Most of the settings for Epic Browser are located in that folder : %LocalAppdata%\Epic Privacy Browser\ ...
So if you need to add direct access for other things, you may follow the same steps provided above.

Regards,
Barb.-
