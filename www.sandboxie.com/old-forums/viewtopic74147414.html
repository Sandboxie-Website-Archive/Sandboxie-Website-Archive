I use SB on Windows 7.

When I use Win Explorer to browse my C drive, it seems to me that I get different results when I use:

- Non-Sandboxed Win Explorer
- Sandboxed Win Explorer

The purpose of this post is to determine whether my SB installation is correct or incorrect.

When I browse the SB folder using a non-SB Win Explorer, this is what I see when I browse to "Program Files (x86)":

C:\Sandbox\<userName>\DefaultBox\drive\C\Program Files (x86)\

In that folder, I see Google and Thunderbird. These are the only two programs that I run under SB. That seems fine to me.

---------------------------------

When I browse the SB folder using an SB Win Explorer, this is what I see:

C:\Sandbox\<userName>\DefaultBox\drive\C\Sandbox\<userName>\DefaultBox

When I click on the "DefaultBox" folder, I get the message:

"Location is not available".

It seems to me that if I look in the following folder, I see all the files that I see in "C" as if I were using a non-SB Win Explorer:

C:\Sandbox\<userName>\DefaultBox\drive\C

I would like confirmation that my SB setup is OK
