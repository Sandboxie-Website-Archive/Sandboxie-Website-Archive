by bo.elam
You cam allow dllhost.exe to run in your browser Sandbox settings. Or, you can close the message and continue with the downloading process, it ll go trough even if you dont allow dllhost.exe to run.

It is normal for dllhost.exe to attempt to run when you browse using File explorer to find the folder where you want a download to be downloaded to. You ll also see dllhost.exe attempting to run like for example, if you want to upload a picture to an image hosting site and you use File explorer while browsing sandboxed for navigating to the picture you want to upload. Another example were dllhost.exe attempts to run would be when uploading a file to a site like Virus total, so, anytime you use File explorer while browsing sandboxed, you might see dllhost.exe try to run.

Keep in mind, if you allow dllhost.exe to run, only dllhost.exe thats in your system would be able to run. If malware using the name dllhost.exe gets dowloaded into rthe sandbox, it wouldnt be allowed to run. So, is safe to allow dllhost.exe to run in Sandbox settings.

Bo
