But if folder a3yov22x.default changed or renamed by firefox, OpenFilePath doesn't work anymore,...
So what you want is to be able to specify -

Code: Select all

OpenFilePath=C:\Documents and Settings\User\Application Data\Mozilla\Firefox\Profile\*\signons.txt
But wildcards are not supported at this time.

You can fake it, though. You could have a small program or script that adjusts Sandboxie.ini according to the actual profile folder name.
