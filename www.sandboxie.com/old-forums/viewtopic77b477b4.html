I'm guessing that since it's a portable version of the program, it's folder location cannot be known in advance.
It can be placed anywhere.
So it's probably necessary to use a wild card for the path.
Maybe "something" like this would work:

[Template_Local_ClamWinPortable]

Tmpl.Title=ClamWin Portable
OpenPipePath=clamwinportable.exe,*\ClamWinPortable\*
OpenPipePath=clamwin.exe,*\ClamWinPortable\*
ReadFilePath=clamwin.exe,*\ClamWinPortable\Data\log\*
Tmpl.Class=Local
