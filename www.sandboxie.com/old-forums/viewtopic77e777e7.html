OK, I see I have to explain every little thing I do, or this discussion will go on.

The point of the non-expanding empty folder has actually crossed my mind while working on this aspect of Sandboxie, believe it or not. And my thought on the matter was that having a folder icon with no plus or minus signs next to it is going to be more confusing, than having an expanding folder icon that shows (None) when you open it. So, I decided to make it this way, and I'm going to stand by my decision. Also, this is too minor a point to make it an option to behave one way or the other.

As for putting the number of files next to the folder. With the current design, the contents of the folder are only scanned when you expand the folder. Which means the number of files can only appear after you expand the folder, and that might look a bit silly. Also, what number should go there? Number of files, of sub-folders, or both? I'm not sure that I would like every folder name to have a trail of text behind it:

+ Downloads (15 files; 4 sub-folders)
