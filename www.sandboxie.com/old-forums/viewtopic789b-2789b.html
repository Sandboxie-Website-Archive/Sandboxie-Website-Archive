Changes in 5.17.6
1) TLS error when loading google sites in IE has been fixed.
2) Excel crash when using data query editor introduced in 5.17.3 has been fixed.
3) Firefox multi-process (Electrolysis) is now supported in Win 7 and later.

There is a FF print to file issue that is still being worked. To workaround this issue, add OpenFilePath=C:\Users\<username>\AppData\LocalLow\Mozilla\temp-*
The other workaround is to disable multi-process:
