I get this error in Chrome: https://i.imgur.com/qbyBSEL.png

It happens in any sandbox, new or old. It will happen in the latest version (5.23.6) as well. I have found out that it's caused by enabling any feature in Sandboxie that 'automatically' launches the application in a sandbox. If I manually right click on a shortcut for Chrome and open it to run inside any sandbox it will allow me to connect to any webpages again. I usually use a pinned Chrome shortcut on my taskbar to launch it, and also enable "Force Google Chrome to run in this sandbox" setting inside the "Applications" settings window.
