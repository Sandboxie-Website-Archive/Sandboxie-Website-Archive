Read the thread posted by Barb, but I ll make it easier for you. For NoScript, allow Firefox Direct file access to "storage-sync.sqlite", that file is where NoScript saves data in Firefox 57. About Ublock. I am not using adblockers as NoScript pretty much gets rid of most of them but I believe in Firefox 57, UBO is saving data in \browser-extension-data\uBlock0@raymondhill.net\storage.js
