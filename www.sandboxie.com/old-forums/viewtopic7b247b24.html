Hello. I have been using Sandboxie for some time now on 4 PCs. I don't run anything in a sandbox that often, I just have Sandboxie installed if I need it. I have my desktop (main gaming PC), a media PC (on all the time), a tablet (for IT house calls) and a gaming laptop. I know Sandboxie runs in the background if the machine is on but I only use it for the machine I'm currently working on if needed.

I just want to know if one license will work for all my PCs without causing a licence issue. Also, can you disable Sandboxie from starting at logon and then start it if I need it? I have an AnyDVD lifetime licence and its installed on 3 PCs. I just disable it from auto starting with Windows so I don't get a licence error. Post by bo.elam » Sun Oct 06, 2013 5:41 pm

You don't need to disable Sandboxie from starting with Windows but if you want to do so, you can in Sandboxie control>Windows shell integration, untick "When Windows start". Sandboxie allows you to have one or more of your computers running at the same time, doing that does not cause License problems or errors.
