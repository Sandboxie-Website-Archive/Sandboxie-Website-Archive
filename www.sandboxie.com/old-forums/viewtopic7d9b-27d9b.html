Hello John connor,

If you want to allow direct access to the folders (meaning allowing Sandboxie to modify those locations on your host), please try this:
https://www.sandboxie.com/ResourceAccessSettings#file

Right-Click on your Sandbox --> Sandbox Settings---> Resource Access --> File Access --> Direct Access
Click Add Program to choose the program that should have access to those locations (in this case, Pale Moon).
Hit Add and select the folders you want that program to be able to access.
Hit Apply and Ok your way out
Configure --> Reload Configuration.

Also, if you are trying to access bookmarks/history, etc , there are templates in Sandboxie that will help you:
Right-Click on your Sandbox --> Sandbox Settings---> Applications--->Web Browser ---> Firefox
Scroll down until finding the Pale Moon options.

Regards,
Barb.-

If so, try removing icons from your Desktop and re-try (see the last response in the thread).

Otherwise, you can add any changes directly to the Sandboxie configuration file:
Configure---> Edit Configuration

For example, to give direct access to the entire PaleMoon profile folder, you can do this:
OpenFilePath=palemoon.exe,%Tmpl.PaleMoon%\
Modify the path after "palemoon.exe," to add a specific location of your choice.
