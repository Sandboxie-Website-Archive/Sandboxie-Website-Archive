The issue with Chrome is the token error. SBIE cannot get access and thus it's denied.

Firefox is also a token issue. It also has an issue with Flash if you have that seperate plug in installed.

The updates were Kernel level. Thus, many many many programs can be affected.

It has nothing to do with SBIE vXXX. If you've installed those updates (Again, please read the notice), SBIE will not work to surf web.
