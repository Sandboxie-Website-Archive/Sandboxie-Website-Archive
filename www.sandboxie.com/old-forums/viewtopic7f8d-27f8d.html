safesurfergir, next time you close IE, click the option to Delete contents.

You should read the Getting started Help guide.
http://www.sandboxie.com/index.php?GettingStarted

You can start here, read about Delete and Recovery settings. Delete settings show you how to set contents of the sandbox to be deleted automatically and Recovery settings are available so you can save files out of the sandbox. Like for example, if you download a video or a picture, you can set the sandbox to give you the chance to save those files to your hard drive.
http://www.sandboxie.com/index.php?SandboxSettings

You should also take a look at Application settings>Web browser>Internet explorer, there you ll see settings that you can tick. You might want to tick the one that allows you to save bookmarks, that way you can save bookmarks while browsing sandboxed. Good luck and don't worry about viruses as long as you dont recover them to your hard drive and execute them.

Is also a good idea, as a first time user, to know how Sandboxie and your antivirus interacts with each other. Read this FAQ.
http://www.sandboxie.com/index.php?FAQ_Virus

Bo
