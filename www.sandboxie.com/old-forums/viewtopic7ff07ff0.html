2 suggestions: Renaming sandbox + Full file path

Posted: Thu Sep 17, 2015 5:30 am
by UserOfSandboxie
Hello,
I have 2 suggestions:

1) Make it possible to change the capitalization of some sandbox. I.E changing the name of sandbox from "Program" to "program" directly without using third name.

2) It will be nice that when sandboxie report an error related to file/folder, it will output the full file/folder path instead of just it's name.

For example:

Instead of printing explorer.exe it will print C:\Windows\explorer.exe


Instead of printing data_2 it will print %USERPROFILE%\AppData\Local\Google\Chrome\User Data\Default\Cache\data_2.

Maybe it will output the related paths in the window and/or when pressing "Copy Contents to Clipboard and Close Window".
This will help finding the problematic file/folder so there is no need to guess where it is or search it by it's name.
