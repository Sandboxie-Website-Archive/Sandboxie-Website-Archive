Hello WhiteSurgeon,

Sandboxie is not a VM, but an isolation program. You can set internet restrictions (all or nothing type of thing) with this setting:
https://www.sandboxie.com/index.php?Res ... nsSettings

Running multiple instance of a program depends on whether the program allows for it or not.
We have many Steam users that use Sandboxie to achieve that goal, however, not every game will allow it.

You can search the forum for specific games and see the latest posts regarding them.
If you let us know what are you trying to do, exactly, we will be able to better assist.
