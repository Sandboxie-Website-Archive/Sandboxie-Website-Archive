Wil70 wrote:Question: can I today modify or copy/duplicate a sanbox from C:\Sandbox\user\DefaultBox, or is this folder private to sanboxie and we shouldn't copy or duplicate things.....
Yes, you can copy sandbox folders from inside C:\Sandbox\user, and save the folder somewhere else in the computer and at a later time, copy the saved folder and place it back inside C:\Sandbox\user. Thats what I do for one particular program which I install in a sandbox that's set not to delete automatically. After installing the program and terminating all programs, I copy the folder of the sandbox where I installed the program and save it to use it at another time.
