DS78 wrote:Everything seems fine with version .12 so far. Only slightly odd thing I've noticed is the "Automatically delete contents" setting now takes longer to invoke when closing the browser for example. Previously it would clear in about 2-3 seconds of closing. After version .12 installed I've seen it take as long as 30+ seconds. Not always, but it did not do this with the previous versions. Nothing serious but I thought I would mention.
I think it's a coincidence probably some background applications or scheduled maintenance that contributed to disk activity. Sandboxie uses "rmdir" which is a MS-DOS command to delete these contents and will invoke when there is no running program in the sandbox. So, basically sandboxie is not causing that delay.
