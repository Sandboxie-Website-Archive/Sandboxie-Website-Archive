A fair bit of success on my part.

Looks like sandboxie isn't playing nicely with Comodo. (This includes with comodo fully disabled but still installed)

Once comodo was uninstalled:

Running application normally still produces the invalid privileges errors below. I'm not sure why sandboxie cant/dosnt elevate its call to RpcSs Service
SBIE2337 Failed to start program: [44 / 740]
SBIE2204 Cannot start sandboxed service RpcSs (740)

Running the applications as administrator works fine. I have tested this with chrome, IE, thunderbird, steam, a few games, evernote, java based apps


So issues found:
- Sandboxie dosnt/cant elevate the call to the RpcSs or the DcomLaunch unless the application is already elevated.
- Sandboxie dosnt work with Comodo internet security installed.
- Sandboxie dosnt work if Malwarebytes real time protection is enabled
- The issues with the above AV's causes the memory errors E5 and the dcom launch error -4


Hopefully this helps someone whos scratching there head wondering why this just isn't working.
