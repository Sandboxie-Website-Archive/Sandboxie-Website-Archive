Re: How to set Sandboxie to prevent Chrome to migrate all bookmarks and settings in the Sandbox?

Posted: Wed Mar 28, 2018 4:41 am
by bo.elam
You need to hide the files inside your current profile. Go to Sandbox settings>Resource access>File access>Write Only access, Click Add program, add chrome.exe. Click Add, navigate to your current Chrome profile and select it. That makes the Profile folder look empty to the installer so it creates new files in the sandbox.

by bo.elam
Palemoon, you need to create a sandboxed shortcut to run Chrome that you installed in the sandbox.

I dont use Chrome, have never installed it in my system, so I am not familar with it but I ll tell you what works for Firefox. I am pretty sure it should work the same with Chrome.

Create the shortcut using one of this two procedures:

1. Open Sandboxie control>Configure>Windows shell integration, Click Add shortcut icons, select the sandbox where you installed Chrome, Click OK, that opens Sandboxies Start Menu. Find Chrome in one of the menus and click it. That click places the sandboxed shortcut at your Desktop. For example, if when you installed Chrome in the sandbox you ticked the option to place a shortcut at the Desktop then you ll find the shortcut at the Destop menu in Sandboxies Start Menu.

2. Using a non sandboxed Windows explorer/File explorer go into C>Sandbox>User, in the folder of the sandbox where you installed Chrome find the Chrome installation. If you installed it to Drives + C + Program Files (86), then look for that folder inside C>Sandbox>User>the sandbox where chrome is installed, find chrome.exe, and make a shortcut. You can do it by right clicking chrome.exe and selecting Send to>Desktop.

Either way should work for creating a sandboxed shortcut.

Or, to avoid using a shortcut, you can run Chrome by using Sandboxies Start Menu (Right click Sandboxies icon by the clock>Hover the browser over the sandbox you installed Chrome>Click Run from Start Menu>Find Chrome in one of the menus, like Desktop and click it). That runs Chrome that you installed sandboxed.
