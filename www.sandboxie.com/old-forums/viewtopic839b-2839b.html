Posted: Sun Aug 25, 2013 2:43 am
by Zer0Kool
Sandboxie 4.05.06 (64 Bit)
CDBurnerXP 4.5.2.4214 (64 Bit)

If you get this message with CDBurnerXP and Sandboxie:

SBIE2205 Service not implemented: SHGetPropertyStoreForWindow

Check that the following Sandboxie setting is enabled

Applications --> Miscellaneous
Permit programs to update jump lists in the Windows 7 taskbar

This should fix this message.
