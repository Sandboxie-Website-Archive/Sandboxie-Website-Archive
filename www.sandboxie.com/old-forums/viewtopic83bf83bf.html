Someone sent me an NTOSKRNL file which I investigated so I might have a solution to this problem. This may or may not work because I could not test it on a running system and could only study the file "at rest." So I'm not making a full release at this time because I am not even sure it works.

For those who would like to give it a try, first you should install version 4.01.06, available here
http://www.sandboxie.com/phpbb/viewtopic.php?t=14453

Then you have to download a replacement SbieDrv.sys file from this link
http://www.sandboxie.com/SbieDrv.sys
- download this into the C:\Program Files\Sandboxie installation folder, then restart your computer.

Thank you goes out to the person who sent me the file!
