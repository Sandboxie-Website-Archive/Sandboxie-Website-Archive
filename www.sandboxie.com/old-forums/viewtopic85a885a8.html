by bo.elam
undrline2 wrote:I just started using Sandboxie, but I'm having issues. I run a suspicious installation program inside the sandbox, everything checks out fine, then I want to install it outside the sandbox.
You can use Sandboxie for testing programs, and keep the installation for a while, that usually works out great. But using Sandboxie for telling if a suspicious program is malicious or not is not a good idea. Some malware can tell when they are in the sandbox and wont do nothing harmful to fool you into running it outside the sandbox.

To get rid of sandboxed installations, deleting contents is all you need to do. No need to run CCleaner.

Bo

Hello apple,

To delete programs installed inside Sandboxie, right-click on the Sandbox ---> "Delete contents".
To manually delete your Sandboxes, go to C:\Sandbox\YourUserName (this is the default location, if you changed it in the past, it may be located elsewhere). Delete the desired folders.

If your browsers are still opening Sandboxed you may have set them as Forced Programs. Please see this entry so that you can check your Sandbox:
https://www.sandboxie.com/ProgramStartSettings#program

The same applies to folders, please see this entry:
https://www.sandboxie.com/ForceFolder

Regards,
Barb.-

I assume that you know to install programs inside of a sandbox by...
1) Create a sandbox to use for the program (Sandbox > Create New Sandbox).

Choose the sandbox which will contain the program, from the list.
