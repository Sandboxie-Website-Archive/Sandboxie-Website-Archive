Palemoon menu not visible on Windows 10 [SOLVED]

  • Quote

Post by ebulerdo » Sun Jan 29, 2017 11:42 am

Hello,

I am using Sandboxie 5.16 64bit in Windows 10 Home. My default browser is Palemoon, running v27.0.3 64bit.

While Palemoon displayed perfectly in Windows 7, I am having a problem since I upgraded to Windows 10. The text in the menu bar is not visible because the text is white and the brackground is white too. This issue doesn't happen when I run Palemoon outside the sandbox. Other users have commented the same problem in the Palemoon forums.

I contacted Palemoon's developers and they said: "Pale Moon needs access to the registry to know what color the accent color is. If sandboxie prevents that, then it won't work."
