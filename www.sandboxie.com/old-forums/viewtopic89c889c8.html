I experimented briefly with IE 10 on 32-bit Windows 8 with this setting:

Sandbox Settings > Applications > Web Browser > Internet Explorer >
Allow direct access to Internet Explorer bookmark and history database

It seems to work as I would expect: Websites I enter into the address bar in a sandboxed IE 10 can
also be seen in IE 10 outside the sandbox. And can still be seen after the contents of the sandbox are deleted.

And if I clear that setting, there is no longer sharing of web site history across the sandbox.

Next I will check in Windows 7 with IE 10.

Post by tzuk » Thu Aug 08, 2013 7:31 am

I think that setting from my earlier post might record only the URLs that I enter directly into the address bar.

Looks like I need two settings for full history:

Sandbox Settings > Applications > Web Browser > Internet Explorer >
Allow direct access to Internet Explorer bookmark and history database
Allow direct access to Internet Explorer cookies

That cookie setting was updated recently in 4.05.03 so you should probably get version 4.05.04 before trying this.

It is still not perfect. Web sites visited don't always appear in the history immediately, sometimes only after restarting IE.
And, it looks like history entries can't be deleted in the sandboxed IE.

I think you will have more luck wiith this after switching to a different browser, if that's an option.
