I found Sandboxie by Googling windows sandboxing programs. Sandboxie is exactly what I wanted and easy to learn. Now I can download files in sandboxie, run my antivirus in the sandbox, and run the downloaded software in the sandbox and see if anything happens. I haven't had any major saves yet, but I did notice that on my routine antivirus scan, the 20 or so tracking cookies that usually pop up are nowhere to be found. The only downside is that the paid version costs too damn much. If there was a standard ( like free/standard/premium) version that cost <$10 and allowed unlimited sandboxes I would buy it.
