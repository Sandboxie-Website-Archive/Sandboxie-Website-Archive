by Curt@invincea
The error in the original post "SBIE2101 Object name not found: , error OpenProcess (C0000022) access=001FFFFF initialized=1" is a complex issue caused, in this case, by delay loading of advapi32.dll. Windows CU changed something with how and when this DLL is loaded and it is creating a race condition causing Chrome to crash. Certain Chrome plugins seem to bring out the problem. We are still looking at this issue.

However, I don't see how this could be fixed by recreating your Chrome profile.

coffeecup25, you were getting the same error msg (SBIE2101) as the original post?
