Post by SnDPhoenix » Sat Apr 05, 2008 1:52 am

is there a way to disable sandboxie from automatically searching for updates etc?
Actually yes, afaik, when you first install Sandboxie (starting with 3.x) it asks you right away if you want to check for updates or not, and if not, then you can hit not now, or even click never so it won't check for updates again. If you don't get the message box, then try opening Sandboxie Control, click "Help -> Check for updates" and see if it brings the message box up.

it would be nice to be able to hide the sandboxie icon from the system tray or password protect sandboxie from settings being viewed/altered
Hmm, well as for the icon, I guess thats a possible option to add to the program. About password protecting Sandboxie, it has been brought up before, and tzuk does not like the idea at all, however, he did add a new option to the "new" Sandboxie, which is EditAdminOnly which should prevent any non-admin accounts from being able to alter the settings. Furthermore, you can also use the ForceDisableAdminOnly option as well if you want to.
i think there used to be a menu item that you could put the names of forced programs into instead of having to go into editing the config file would be nice to have
Well what you can do now (with the new Sandboxie) is right click the sandbox you want to edit the settings of, and just select "Sandbox Settings", then from there you can go to "Force Programs" (or ForceFolder) and force a program from there.
