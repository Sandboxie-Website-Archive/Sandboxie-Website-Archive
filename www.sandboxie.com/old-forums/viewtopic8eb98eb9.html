It seems to get stuck on LICENSE.TXT because it is the last one in a set of files If any file in the set failed, the installer asks to retry the entire set. Maybe your anti-virus is keeping the files or locked or something. You might try a tool like Process Explorer, there is a find command that will let you see which process is holding onto some files. You should look for SbieDll or SbieSvc.

Post by desmith1962 » Wed Nov 27, 2013 11:43 am

I am still experiencing this problem when trying to update Sandboxie. I did however find the culprit and a way to get passed this error. I found that it is exporer.exe that is causing the problem. I tried a few programs like "Unlocker" http://www.emptyloop.com/unlocker/ and while Unlocker was able to tell me that the files trying to be updated were "locked" by explorer.exe it failed to "unlock" the files for me because the installation would still get stuck on License.txt. Today I tried to update to the newest beta and the problem happened again. This time I didnt mess with trying to "unlock" any files. What I did was start task manager and then I terminated the process explorer.exe. My desktop went blank except for the installation of Sandboxie that was waiting at the prompt "retry or cancell". I hit retry and the installation completed correctly. Then ctrl-alt-delete to bring up task manager and restarted explorer.exe. While I still dont know why explorer.exe is being such a pain in the ass atleast I can get around it now and finish the updating of Sandboxie when needed. [/b]
