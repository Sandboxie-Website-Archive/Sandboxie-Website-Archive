Hello chris87,

You can try this:
Configure --> Edit Configuration
Paste the following at the end of the desired Sandbox:
ClosedFilePath=firefox.exe,%Desktop%\
Save it.
Configure--> Reload Configuration

That should block files from the Desktop from being dragged into Sandboxed Firefox. You can tweak it to add other locations. More info below:
https://www.sandboxie.com/ClosedFilePath
