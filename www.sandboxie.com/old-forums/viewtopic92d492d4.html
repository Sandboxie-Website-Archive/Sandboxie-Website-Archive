The DMP shows exactly where the error is coming from. But I don't see how Sandboxie is getting into this state. I assume you switched to 5.23.3 because of the Meltdown KB.
  1. Were you using 5.22 before? Did this ever happen in 5.22?
  2. Did it immediately start happening after installing 5.23.3?
  3. Does it happen every time?
  4. Is there another machine you can test on?
