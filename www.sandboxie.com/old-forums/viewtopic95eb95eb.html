See if this helps
Adjust the settings using the non-sandboxed Firefox 57 in about:config to the following:
security.sandbox.content level 2
browser.tabs.remote.autostart true
browser.tabs.remote.autostart.2 false
accessibility.loadedInLastSession false

You can check in about:support to see if multi-process windows is enabled
You may have to delete the contents of some sandboxes if you experience the tab crashes
