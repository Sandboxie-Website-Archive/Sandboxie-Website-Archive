Hello nolek,

Do you have UAC disabled on your system? (If yes, re-enable and reboot, then check again).

Otherwise, is there a chance that Sandboxie files are set to run as Administrator in compatibility mode?
To check, navigate to your Sanboxie installation folder (C:\ProgramFiles\Sandboxie...) and see if the SbieCtrl.exe and Start.exe have any compatibility settings added (right click---> Properties --> Compatibility---> Change settings for all users) .

Also, if you are running Win 7 x64, you will need to download the latest beta to avoid issues with the latest kernel security updates:
