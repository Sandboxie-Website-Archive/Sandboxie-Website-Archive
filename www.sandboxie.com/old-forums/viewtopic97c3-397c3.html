Is Sandboxie compatible with Ubuntu ?
Sandboxie is not compatible with Linux OS.

There was a claim by Wine that would allow Sandboxie to be emulated in Linux. It does not work. As SBIE needs to install a driver and that cannot be done in Wine (much like SBIE.) I've spoke to their Dev and have confirmed this.

Sandboxie is Windows Only. Unless you're in a VM...then you can load Win in Linux..i.e Virtual box or using Boot camp on a Mac. You can also just run Ubuntu Live. Either USB / CD. But again, SBIE will only work when installed on a Windows based kernel.
