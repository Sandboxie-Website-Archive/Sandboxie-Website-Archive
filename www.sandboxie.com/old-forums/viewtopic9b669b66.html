Sandboxie version 2.78.5 Released

Posted: Wed Feb 14, 2007 3:39 pm
by tzuk
Download links:

Sandboxie version 2.78.5

Sandboxie 2.78.5 for 64-bit Windows

Hopefully this version should resolve most of the current technical problems with Sandboxie.

Next beta version will focus on changes to Sandboxie Control.

Changes in version 2.78.5:

* Message 2209 (SYSTEM key) removed

* Internet Explorer should not keep running when all its associated windows have been closed

* Sandboxie Control now starts the Sandboxie Service, if it is not running

* Sandboxie Service initialization further improved for Windows Vista

* More bug fixes

Installer Changes:

* Installer will stop Sandboxie Control

* Uninstaller does not show license agreement page
