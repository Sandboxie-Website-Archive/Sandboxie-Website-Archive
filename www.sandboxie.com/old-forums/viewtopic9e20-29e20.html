If you still have problems after 5.06 RC, you can send us information, you would need to do the following steps (step 1 at least)

Step #1 You can use procmon and post the results to us.
Procmon: https://technet.microsoft.com/en-us/lib ... 96645.aspx (free.)
Close any programs that are not needed:

Run Procom (not sandboxed) and then try and fire up FF (sandboxed of course). Let the error happen or it not load, stop capturing events in Procmon (under file) and then Save the results (under file, make sure "save all events it ticked") and save in PML format. These files can be large, so post somewhere we can download and look. (Please zip them!) and link to Support@sandboxie.com

Step #2: If you can give us a snapshot of your installed software by opening Powe Shell (as admin) and type:

Get-WmiObject -Class Win32_Product | Select-Object -Property Name > C:\Software\PCapps.txt

The PCapps.txt will be created listing your software, file you can send to us as well as the PML
