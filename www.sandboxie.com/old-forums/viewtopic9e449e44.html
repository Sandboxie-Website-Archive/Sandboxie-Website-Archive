Using sandboxie as Docker

Posted: Thu Mar 12, 2015 1:39 am
by nazcasistemas
Have you thought on making Sandboxie to be like a docker system for windows?
Seems like you already have a lot of the features that can help in this use case:
- Sandboxie separates the sandbox from the rest of the system and allows to install apps inside the sandbox (like a docker container does).
- The user could package applications including registry/windows changes made inside the sandbox in redistributable zip files.
- Have a filesystem jailed from the main windows installation.

I have read that some complex programs have issues with sandboxie (SQL Server) but maybe this use case could attract some funding to the Sandboxie developers to improve it for this type of apps.
