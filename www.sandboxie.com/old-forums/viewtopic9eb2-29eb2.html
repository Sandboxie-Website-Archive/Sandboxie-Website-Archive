MrBoxer wrote:Chrome 64 bit ( 40.0.2214.115 m ) bookmarks won't save in sandboxie ( 4.16 64 bit). I already allowed " direct access to bookmarks/bookmark & history database.
This can happen if your Chrome profile folder is not where the Sandboxie templates expect it to be:
%Local AppData%\Google\Chrome\User Data\Default

That's:
C:\Users\(username)\AppData\Local\Google\Chrome\User Data\Default

If the "Default" profile folder isn't there, you must tell Sandboxie where it is:
Sandbox Settings > Applications > Folders
If Chrome is listed in the box with the pull-down arrow, you must Browse to and select the "Default" folder using the Alternate Location box.
(Sorry I can't be more specific, but the Win 8 PC that I'm using doesn't have anything in the box with the pull-down arrow, so I can only assume that you will see a Browse button there)
