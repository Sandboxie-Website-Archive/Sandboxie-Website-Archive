When you install a program in a sandbox, it can think that it is adding a shortcut to the Start Menu or to the user's Desktop.
Those shortcuts will actually be saved in the sandbox.

See if there's a shortcut to the program there:
By right-clicking the Sandboxie Control tray icon, hovering the mouse over the sandbox name, and then click on 'Run From Start Menu'.
On the menu that opens, just look to see if there is a shortcut to the program - either on the Start Menu or on the Desktop.

If there is, then just close that menu and use the Sandboxie tray icon's menu:
Configure > Windows Shell Integration
Click the "Add Shortcut Icons" button
http://www.sandboxie.com/index.php?ConfigureMenu#shell

Pick the sandbox containing the program and then pick the program's shortcut from the menu you looked at before.
A shortcut will then be created on your real Desktop.
Try it, to see that it works, and then move it to the Startup list.

If the program did not add a shortcut to the Start Menu or Desktop in the sandbox, a shortcut can still be created manually.
Let us know if you need help with that.
