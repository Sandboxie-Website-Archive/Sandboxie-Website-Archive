> Windows Defender
> Voodoo Shield (Paid)
> Zemanan AntiMalware (Paid)
> Sandboxie (Paid)
Zeema Security products may cause issues, this is noted in this sticky here http://forums.sandboxie.com/phpBB3/view ... 11&t=21539 as well as other security products. Windows Defender is known to be ok.
> My question is why 5.11.7 version and lower versions of Sandboxie version work.
> And NOT 5.11.8 Version?
Again, works just fine in all tests (no security products and very limited addons, extension...as we normally test only the CORE browser).
*Remove problematic 3rd party security products
* and FF browser addons/extensions.
Then use FF. I'm certain it will work. This is why we have a list of known issues with 3rd party security products. Simply disabling some is not enough sometimes.

You can run Resource Access monitor and/or Procmon to get us a log, as we're not seeing it with the above criteria.
You can do a ProcMon that will show us if there is a problem:
Get procmon here (free) https://technet.microsoft.com/en-us/sys ... 96645.aspx

Steps
Close any programs that are not needed.
Run Procom (not sandboxed)
and then try and fire up FireFox (or the browser causing issues) (sandboxed of course).
Run your process to show that shows the issue you are receiving
stop capturing events in Procmon (under file)
then Save the results (under file, make sure "save all events it ticked") and
save in PML format.
These files can be large, so post somewhere we can download and look (link)
OR (Please zip them!) and send to Support@sandboxie.com
