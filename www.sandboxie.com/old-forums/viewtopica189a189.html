Hello __jhc,

You are missing entries in your Sandbox configuration file. Please create a new Sandbox and see if you can run programs via that Sandbox (if yes, copy the settings from the new sandbox under the DefaultBox).

If the issue persists, uninstall Sandboxie without keeping your Settings (you will be asked during the uninstallation process "Would you like to keep your Sandboxie.ini configuration file? Select "No". ) . Then, re-install it (be sure to use the latest version 5.22 or the latest beta 5.23.5).
Latest beta can be found here:
viewtopic.php?f=64&t=25114

If you are still experiencing issues after following those steps, please provide this information:
What's your Sandboxie version?
Is there any antivirus involved?
Are there any security policies in place? (GPO, applocker, etc )/

Regards,
Barb.-
