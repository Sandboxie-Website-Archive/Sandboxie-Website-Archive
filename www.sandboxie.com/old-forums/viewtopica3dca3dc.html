by tzuk
I'm not sure that I follow your explanation. In my experience with proxy software -- which I admit is very limited, and was always in the context of looking into a compatibility problem with Sandboxie -- once the proxy software is started, the computer changes to a new IP address and Internet connectivity is routed through the proxy service.

For this reason it doesn't make sense to me that you would be able to run more than one proxy service at the same time. At least not with Sandboxie. You might need to look for a tool that lets you have multiple sets of network routing tables on the same network interface. If such a tool exists.
