Combined 32/64 installer:

Separate:
http://www.sandboxie.com/SandboxieInstall32.exe
http://www.sandboxie.com/SandboxieInstall64.exe

Sandboxie 5.24 has been released
Change log: https://www.sandboxie.com/VersionChanges#v_5.24

Code: Select all

 [b]All beta 5.23 changes [/b]

[b]Changes in 5.24 [/b]
This is the Release Candidate version.  

[b]Changes in 5.23.7 [/b]
Added support for Secure Boot. 
See changes in 5.23.6 for more details. 

[b]Changes in 5.23.6  ---> Not signed for Secure Boot - If you have Secure Boot enabled, please do not download this build. Use 5.23.7 instead)[/b]
1) Fixed BSOD with  Print2Flash install.
2) Fixed BSOD in Win 10-32 CU with KB4074592.
3) Fixed Win 10-32 FCU dllhost.exe / sbiedll.dll initialization fail.
4) Fixed crash in Windows 10 FR Build 17035 and added support for FR up to 17101 .
5) For x86 systems --> If you encounter "SBIE2335 Initialization failed for process Start.exe [11 / 87]" after creating a new Sandbox/installing this beta, please allow 30 seconds to go by, then re-try the action. This is being investigated. 

[b]Changes in 5.23.5  ---> Secure Boot Support added [/b]
Fixed certificate issues affecting build 5.23.4 
See changes in 5.23.4 for more information.
Combined 32/64 installer:
http://www.sandboxie.com/attic/SandboxieInstall-523-5.exe
Separate:
http://www.sandboxie.com/attic/SandboxieInstall32-523-5.exe
http://www.sandboxie.com/attic/SandboxieInstall64-523-5.exe

[b]Changes in 5.23.4 (reverted until certificate issues are fixed)[/b]
Fixed unable to change Windows 10 desktop background (https://forums.sandboxie.com/phpBB3/viewtopic.php?f=11&t=25291)
Fixed Office 2013-C2R failure to start.

[b]Changes in 5.23.3[/b]
Fixed Win 7-64 Sbie driver load failure with KB4056897 /KB4056894 

[b]Changes in 5.23.2[/b]
Source code updated to Visual Studio 2015.

[b]Changes in 5.23.1[/b]
1) Added support for Firefox Favicons (Right-click on your Sandbox --> Sandbox Settings--> Applications ---> Web Browser ---> Firefox --> Allow direct access to Firefox bookmark and history database).
2) Addressed an issue in which Chrome v64 dev/Canary were crashing when Sandboxed [https://forums.sandboxie.com/phpBB3/viewtopic.php?p=130933#p130933]
