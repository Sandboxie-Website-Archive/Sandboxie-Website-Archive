If you want the "Run as UAC Administrator" check box setting to persist. Sandboxie does not save this setting

Example:
If the application is set to run as admin in its Compatibility properties tab, Sbie will check this box automatically. You can also add the /elevate command line switch to the start.exe command. E.g. "C:\Program Files\Sandboxie\Start.exe" /elevate /box:DefaultBox "C:\Users\Public\Desktop\Acrobat Reader DC.lnk"
