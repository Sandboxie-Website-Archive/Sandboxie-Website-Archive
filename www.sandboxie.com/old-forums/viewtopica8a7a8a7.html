Re: BITS Client Error 0x80070005 when Chrome in Sandboxie [SOLVED]

Posted: Tue Jan 30, 2018 2:59 pm
by Barb@Invincea
Hello iceberg,

I reached out to the devs and received the following information:

Regarding BITS, Chrome is the only big app that uses it along with Windows update. However, these updates will not work inside Sandboxie.

SandboxieBits.exe will work in Windows 7, that is expected and it explains why the command bitsadmin / list does not return an error. BITS will not work in SBIE past Win 7 (so 8.1 or 10 will trigger an error message).

At the moment, Bits' behavior is not a priority, therefore the devs are not looking at changing/modifying it.

Regards,
Barb.-
