Sandboxie version 2.31 Released for Testing

Posted: Thu Mar 23, 2006 3:27 pm
by tzuk
I expect this version to be stable, but it can't hurt for more people to test drive it for a while.

Updates:

Full support for installation of MSI packages through Windows Installer

Configuration for OpenFilePath and OpenKeyPath is still available only by manually editing Sandboxie.ini, but will no longer be overwritten when making other configuration changes through the Sandboxie GUI.

Version 2.3 introduced program delays to keep the system from locking up under rare conditions. These delays had a noticable negative effect on usual performance, and have been fine tuned in this release.

Fixed a bug that was crashing the Yahoo! Messenger.
