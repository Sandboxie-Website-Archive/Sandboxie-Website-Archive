Hello cryptogram,

Sandboxie includes templates that point to the default locations where programs save their data, in order to allow them to properly work Sandboxed.
If you have installed Thunderbird in a location other than the default one, you will have to manually give Sandboxie access to it. Otherwise, the template should work.
Are you experiencing any issues?

To access Local Appdata via Windows click on Start and type %LocalAppdata%
