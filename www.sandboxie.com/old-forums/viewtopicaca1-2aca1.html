LAUNCHER VERSION: 2.3.0.4
INSTALLER VERSION: 1.0.1.2

PLEASE READ THE README.TXT DURING INSTALLATION FOR INSTRUCTIONS

** DO NOT USE on a system with a locally installed version of Sandboxie!!


<-- NOW packaged as an installer!! -->

SandboxiePortable now uses the official Sandboxie installer. This has several benefits. Installation is now much simpler, you don't have to manually extract anything. Since the installer cleans up everything upon removal, you can now use the shell / sendto extensions portably.

I've also included an Updater utility to simplify updating to a new version of Sandboxie. It is located in the same directory as the launcher. A separate download link is below for users who do not need to update the whole package. Place the Updater in the same directory as the launcher.


** NOTE TO USERS UPGRADING FROM 3.01.21 or 3.02 -

The new GUI versions of Sandboxie use a new layout for the INI file. DO NOT use your old INI file directly with this new portable version of Sandboxie. Run the launcher once and let it copy a default INI to the application directory, then modify your settings via the GUI.

LINKS:

(MD5: a7de3f694227bbbafdb4d0b1f95c6546)

SandboxiePortable Updater (included in full package above)

Changelog:

v2.3.0.4

- Installation and removal of Sandboxie now handled by official Sandboxie installer

- CrashCleanup utility updated accordingly

- Shell Integration can now safely be used

- updated README and installer


v2.2.2.3

- Sandbox location now read from FileRootPath setting in the GlobalSettings section for compatibility with new Sandboxie behavior

- A warning will be issued if the deprecated 'BoxRootFolder' setting is found in your INI

*** WARNING ***
For SAFETY you MUST set your sandbox location through the Sandboxie GUI - Sandbox -> Set Container Folder


v2.2.2.2

- Fix: Splashscreen was not displayed if SandboxiePortable.ini was not in launcher folder


v2.2.2.1

- Updated INI processing for compatibility with the new Sandboxie GUI (v3.02.25+)

- Launcher will automatically copy a default INI file to the program directory if one does not exist

- Added a new INI file to contain new Eraser and splashscreen options; settings are no longer stored in the main Sandboxie.ini file

- Updated README with new information

- Updated Sandboxie LICENSE

This was actually covered in the middle of that 22 page monster somewhere

Sandboxie's license only allows you to register it on computers that you own. To comply with the license (which you agree to by downloading and using the portable launcher) SandboxiePortable will not extract/backup/restore any license information.

Though it was talked about, there is not yet any portable licensing available or planned.
