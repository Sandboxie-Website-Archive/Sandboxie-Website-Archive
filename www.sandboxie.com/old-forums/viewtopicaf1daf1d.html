Post by harlan101 » Tue Jun 09, 2015 3:18 pm

SOLUTION:After checking microsoft updates I found that update KB3057839 was the culprit.I uninstalled this update and all is well. Microsoft support web site states
"MS15-061: Vulnerabilities in kernel-mode drivers could allow elevation of privilege: June 9, 2015". The summary states "This security update resolves vulnerabilities in Windows. The most severe of these vulnerabilities could allow elevation of privilege if an attacker logs on to the system and runs a specially crafted application. An attacker could then install programs; view, change, or delete data; or create new accounts that have full user rights. " Hope this helps anyone who has had this issue.

@ Bellzemos You can either wait for a new beta version of Sandboxie or to uninstall a Windows update go to control panel--programs--view installed updates--right-click the update you want to uninstall and select uninstall.I'm sure that Invincea will have a fix soon.If you uninstall the windows update involved it will most likely show up again when windows searches for updates so be aware and don't re-install until a fix is available.You can choose to hide the update until then if you wish.I don't encourage uninstalling windows updates at all.The copy/paste/clipboard is something that I use extensively everyday. Therefore, I uninstalled the windows update for that reason.When Invincea has a fix I will re-install the update.Hope this helps.
