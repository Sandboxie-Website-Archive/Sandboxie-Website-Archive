Post by Barb@Invincea » Thu May 25, 2017 12:58 pm

Hey Dun,
What can I do to find out the cause?
Since the issue happens randomly, you need to start narrowing it down.
Here are some suggestions:

1- Start disabling your extensions and see if the issue stops.
2 - Test a new Sandbox with default settings.
3 - Create a new Firefox profile and see if the problem occurs there.
4 - Try to see if certain websites are more likely to trigger the issue than others .
5- Randomly, open the Resource Access Monitor, see if you can capture the error.
6- If you have an antivirus, disable it and/or remove any browsers protection.

Once you have more info, please start a new thread so that we can better assist you.

Regards,
Barb.-
