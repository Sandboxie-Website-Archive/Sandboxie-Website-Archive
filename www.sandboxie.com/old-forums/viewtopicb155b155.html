Post by Barb@Invincea » Wed Feb 22, 2017 2:31 pm

Hello henryg,

Maybe this can help you (not sure, I still cannot repro the problem):

Close your Sandboxed applications and Sticky Password/Firefox (if running outside Sbie)
Create a new Sandbox
Right-click on your Sandbox -->Sandbox Settings ---> Resource Access --> Windows Access
Hit Add and enter the following:
$:spUIAManager.exe
Hit apply.
Go to Applications --> All Applications
Scroll down , double click on Sticky Password (if not enabled already).
Hit apply, hit Ok.
Go to Configure --> Reload configuration
Test sticky password functionality (if you receive any errors, try the same steps minus enabling the Sticky Password template).

Let us know the outcome.

Regards,
Barb.-
