Isolation/sandboxing works best when you separate programs not only from the system but from each other as well. Using only the DefaultBox, programs you run sandboxed, run isolated from the system but not from each other. Using separate dedicated sandboxes for different programs allows you to really maximize isolation. And also to tailor sandbox settings according to the dedicated program or special purpose of the sandbox. To get the most out of Sandboxie, you have to do this. No maybe . I think you have a Sandboxie license, start doing it.

Regarding how Sandboxie does to control things when you are using 10 or 15 sandboxes? When you force programs to run sandboxed in their own dedicated sandbox, it gets done automatically with no thinking on your part being required. It works seamlessly. The end result for me personally is that I use my computer pretty much as I would if I was not using SBIE with the difference being that I am protected every time I run a file or program in the computer.
