Post by Barb@Invincea » Thu Nov 02, 2017 2:53 pm

Hello maginu ,

The responses to your first two questions are already covered in the thread. Please read all the posts for more information.
Full support for Windows 10 FCU started with Sandboxie 5.22. So you will need to upgrade to it in order to use that environment.
Can any expert explain what had happened which resulted such "leakage" of chrome with sandboxie?
A change in a Windows 10 FCU API caused the issue. It did not only affect Chrome, but any other program that would use that API. Sandboxie had to be updated to process those new changes and properly sandbox them.
The changes were added in Sandboxie 5.21.6 BETA, tested through Sandboxie 5.21.7 Beta, and then added to Sandboxie 5.22 official release.

If you find any specific issues, please let us know.

Regards,
