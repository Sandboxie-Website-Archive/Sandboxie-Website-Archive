Sandboxie version 2.43 Released for Testing

Posted: Sat Jun 03, 2006 12:24 pm
by tzuk
Fixes:

Refined memory usage.
Fixed a small potential security hole.
Improved audio quality in sandboxed programs.
Now plays MIDI audio files inside the sandbox.
Fixed problem when Protected Storage service is stopped.
Now correctly moves files from a protected sandboxed folder to an "open" older.
Windows Sound Recorder can now start inside the sandbox.
