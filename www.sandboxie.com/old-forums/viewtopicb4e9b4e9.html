Hello,

For the last few days, I have been getting these popups everytime when I launch Opera browser (sandboxed):

Popup from Sandboxie:
SBIE2224 Sandboxed program has crashed: opera.exe [DefaultBox]
Popup from Windows(?):
opera.exe - Erreur d'application ; "L'exception unknow software exception (0xc0000409) s'est produite dans l'application à l'emplacement 0x011ffdab. Cliquez sur OK pour terminer le programme.
This issue did not start to occur following any program update. I had been using Opera version 51.0.2830.40 with Sandboxie 5.23.3 for a while without any problem.

Here is what I did so far to try to resolve the issue:

1. I completely uninstalled Opera (e.g. deleting user settings) and reinstalled the newly released version 51.0.2830.55

2. I uninstalled Sandboxie (deleted user settings) and upgraded to version 5.24 (release candidate)

At first, it seemed to have resolved the issue, but as soon as I rebooted my PC, the same issue came back again.


Additional information:

O.S.: Windows 7 x64

Antivirus: Webroot SecureAnywhere 9.0.19.43 (I have been running this build for several weeks without any issue with Sandboxie)

Any other info or logs I can submit?

Thanks

p.s.: This issue does not occur with the most recent versions of Firefox (version 58.0.2), Chrome (Version 65.0.3325.146), and Internet Explorer (version 11.0.96.00.18920)
