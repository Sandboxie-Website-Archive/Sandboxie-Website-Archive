This question should have been posted in the Quick Questions section of the forum.
Sandbox Settings > Delete > Delete Command
allows you to use Eraser v5 or Eraser v6, if you first download and install one of them first.
Personally, I do not recommend SDelete v1.6 or v1.61
Eraser v5.8.8
Eraser v6

I use 5.8.8, and the "eraserL.exe" file that Sandboxie wants me to locate and select, is in my Windows\System32 folder. If not there after the install, then look for it in Eraser's program files folder.

For Eraser v6, locate "eraser.exe" in Eraser's program files folder.
