Sandboxie version 2.33 Released for Testing

Posted: Sat Apr 15, 2006 4:28 am
by tzuk
Updates in beta version 2.33, over last beta version 2.31 :

* Support for complex device names such as those used by dynamic hard drives and RAM disks.

(Technically speaking, support for any device name, rather than only \Device\HarddiskVolumeX .)

* Fixed a few well-hidden memory corruption problems.

* Installs "Run Sandboxed" shortcut for MSI Packages

Download:

http://www.sandboxie.com/2.33/SandboxieInstall.exe

http://www.sandboxie.com/2.33/SandboxieInstall64.exe
