Hello IDontKnowEverything,
"Is it possible to install a program (or a game) within the sandbox, which is already installed on the computer? "
Yes, provided the program can be installed inside Sandboxie.
Also, you can run Steam Sandboxed (for example), block it from seeing your currently installed games, and then select a new location to install your Sandboxed games.
Here's a thread that might help you set this up:
http://forums.sandboxie.com/phpBB3/view ... ilit=steam
Is it as simple as opening an explorer window (within sandboxie environment), finding installation files and execute the (e. g.) setup.exe, to install a new program?
Yes. You can also find the installation file, right-click on it --> Run Sandboxed, or just drag-drop it in Sandboxie Control.
Which path will be default for installation be given by the setup program? And which installation path am I supposed to use when installing in sandboxie?
You can choose any paths you want. It would be the same as selecting it on the host (but it will be created inside Sandboxie) .
I suppose that when I delete the sandbox I used, all the installed programs, etc., will be wiped from my hard disk, right?
Correct. Here's more information regarding how the delete process works inside Sandboxie:
https://www.sandboxie.com/index.php?Del ... gs#command
https://www.sandboxie.com/index.php?SecureDeleteSandbox
