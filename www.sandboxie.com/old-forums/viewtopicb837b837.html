coffeecup25,

I always do a clean install on every major Windows upgrade which is the best way without running any problems. I know it is a hassle and wasting time for reinstalling all your apps, configure settings, and move all data back but everything works great and smooth after doing that. I hate the upgrade process since it will breaks something along the way which I had ran into in the past.

As for Windows Refresh, it is a hit or a miss which is not like a clean install. It might fix some problems or it might not. There is an option to keep all your personal data if you go that route. However, all your software or apps will be deleted. The "C:\Users" folder will be untouched which means that everything inside that folder will be retained. Make sure you select keep your data if your reset or refresh your OS if you don't want to loose all your data.

Always backup all your important data to an external drive, preferably SSD since it doesn't die on you unexpectedly. I had an external HDD which died on me and I lost all my data. Don't make the same mistake like I did.

Regards

Try rebooting your system.

If that does not help, try disabling Chrome's sandbox:
Create a new Chrome shortcut on your Desktop (or desired location)
Right-click --> properties on the newly created Shortcut
Update the target to reflect the following: " --no-sandbox " at the end (no quotation marks)
This should be added after the quotation marks in the Target area, leaving 1 space
