Hello danicx,

I donwloaded PotPlayer and set it as default in Win 10. I received about 20+ notifications from Windows stating "An App caused a problem with the default app setting..." and it reset it Movies & Tv, or Grove Music.
I manually set PotPlayer to default, and then followed your steps but couldn't find "Common settings". I did find General --> Multiple instances ---> Disable: Play files in existing PotPlayer instance, so I used that instead.

I can see the problem, it does not happen if I set PotPlayer as forced instead of using a forced folder. So that may be the best option for you right now.
You can also just highlight all the files and drag them to the desired sandbox as another alternative.

I have made the devs aware and I will update this thread if new info becomes available.
