Abdullah wrote:BTW, I tried KAV last year and uninstalled it ( even KAV removal tool didn't detect any traces ) , but it is still exist in the SBIE compatibility list suggestions after uninstalling/installing SBIE !

Is it something in SBIE or I have to remove some KAV traces manually ?
Abdullah,you can delete the KAV Software compatibility settiung from your Sandboxie config file. Go to Sandboxie control>Configure>Edit configuration, Delete the Template for KAV under Global settings.

I haven't used many programs for which I tick Software compatibility settings but I remember one program in particular, that after uninstalling the program, the Software compatibility setting still showed up in Sandboxie control. I dont recall if I got rid of it the way I wrote above or another way. But you can safely remove it by deleting the Template line. I don't believe the template remaining means that you still have remnants from KAV. Probably something remains but I don't think that has anything to do with the template still showing in the Software compatibility window.
