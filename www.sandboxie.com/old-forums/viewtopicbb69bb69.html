If you have registered Sandboxie yu should be able to run two instances simultaneously in two different sandboxes. It's unlikely that you'll be able to install the game in the sandbox as it could almost definitiely have DRM/anitcheat drivers. So you'll have to have it installed outside, but you will also need: http://www.sandboxie.com/phpbb/viewtopic.php?t=4885 Block Process Access to insulate the anticheat component from the other instance (if it does exist). Jjust be patient until you get the game and try everything out then report back if you have problems. Without having it its just a shot in the dark.

Post by MichaelS » Thu Apr 26, 2012 2:44 pm

Only the focused application/window has input priority. You can't control two windows at the same time with the same input devices. There are games that recognize multiple input devices and allow them to be used simultaneously (2 pointers on the screen), but I don't think Diablo 3 is one of them.
