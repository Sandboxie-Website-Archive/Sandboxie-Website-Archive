The crash of Windows Explorer described in this topic seems to be caused by some incompatibility with avast.

I fixed it by updating the avast compatibility settings to block the avast DLL (snxhk.dll) for the Windows Explorer process.

It seems to work well enough, please let me know if you encounter any problems with this.
