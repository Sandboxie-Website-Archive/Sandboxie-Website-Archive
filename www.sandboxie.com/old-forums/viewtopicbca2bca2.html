fcharat wrote:This problem has only occurred since the last month. It also occurs when I first open the sandbox after deleting it. I believe it happens less if I reboot the computer.
FF run within the SB will use the settings that are present on your host. Also, you should make certain that FF is set as your web browser in Applications in SBIE.

If you set google to be your homepage in FF outside the sandbox, it will persist when run in the SB (you have to make certain that your contents are deleted in that SB routinely.)

Not sure what help uninstalling / reinstalling will do..other than cause a headache. If you were getting errors that indicated that, maybe. But this is a setting, cookie, stale SB content issue.
