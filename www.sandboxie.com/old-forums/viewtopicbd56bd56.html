Posted: Thu Jul 14, 2011 2:34 am
by NOTT
Hi,

I realise that ACL for folders & files created in the sandbox are currently fixed at 'Authenticated Users' and current user.

I am setting up a PC with multiple user accounts mainly for web surfing. The browser will always be run in a Sandbox. Before putting in Sandboxie, each user's files are protected by ACL of the accounts. However with Sandboxie, a user will be able to look at other users' sandboxed files.

This is also mentioned in another thread -

http://www.sandboxie.com/phpbb/viewtopi ... 820adff175

What is the rationale behind giving access to Authenticated Users?

Is it possible to change the ACL to 'Administrators' and current user?

Thanks!

Posted: Thu Jul 14, 2011 7:13 am
by tzuk
The rationale is to try to make it easier to access files in the sandbox from any account, and get rid of them easily if you need to. You're actually looking to have it the other way around. But perhaps I can add such a sandbox setting that selects between Authenticated Users and Administrators for the files in the sandbox.
