Hello,
I use version 4.1.04 in 64 bits. I put in my Windows Media Player "folder forced." When I launch a video I double clicking the following message appears: "SBIE2314 abandonment process wmplayer.exe (HR = 8007000E / 2)".

Posted: Sun Mar 31, 2013 4:24 pm

Instead of forcing the folder, try forcing "wmplayer.exe" to run sandboxed:
Sandbox Settings > Program Start > Forced Programs > Add Program > wmplayer.exe

Posted: Sun Mar 31, 2013 5:05 pm
by Mandrake
this is what I did as the setting. With the old version 3.76 there was no such worries.

I solved the problem by installing another media player and putting in "programs forced." The problem is therefore between Windows Media Player and version 4 of Sandboxie.
