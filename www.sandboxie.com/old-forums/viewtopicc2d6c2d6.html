How do I arrange it so that Sandboxie does not automatically start when I boot the PC? Specifically, I have disabled the "Start Sandboxie Control" when Windows starts, but when I run certain programs, they are Sandboxed. Hello cocoon,

If you have a forced program, it will invoke Sandboxie when you try launch it. The Sandboxie service is involved and turning it off will cause the program to error out (as it won't be able to get to Sandboxie).
Your options are to remove that program from the forced programs list, run the program outside Sandboxie or temporarily disable forced programs:

To run a forced program outside Sandboxie, right click on the program --> Run Sandboxed --> Run outside Sandbox
OR
Hold ctrl + shift , right click on the application --> Run Sandboxed.

To disable Forced Programs:
Sandboxie Control --> File ---> Disable Forced Programs
Select the amount of time (in seconds) that you'd like to disable them for.
Once you have set the time, you can right-click on the Sandboxie Control icon in the taskbar and select "disable force programs".

Barb.-
