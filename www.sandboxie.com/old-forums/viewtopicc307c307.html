Is anyone of you could post here your access logs? Just follow procedure below.

1. Create New Sandbox. Copy Settings is None. (Sandbox Menu -> Create New Sandbox)
2. Terminate all programs in the sandbox. (File Menu -> Terminate All Programs)
3. Make sure no program is running in the sandbox(There should be no list of processes in Sandboxie Control Window)
4. Open Resource Access Monitor. (File Menu -> Resource Access Monitor)
5. Run Firefox as sandboxed in the newly created sandbox.(Right-click Firefox Shortcut -> Run Sandboxed)
6. Click Copy Contents… from Resource Access Monitor window
7. Post here the logs in code tags
