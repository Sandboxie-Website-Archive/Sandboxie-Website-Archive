SBIE & Chrome x64 Random Errors [SOLVED]

Posted: Thu Feb 11, 2016 8:56 am
by Craig@Invincea
FIXED in Beta 5.07.8

Random crash my be generated when using Chrome v48/49 x64 bit only.

Chrome is using the same hook as Sandboxie. And thus, when they get together, you will receive this error below. (Will have 888,999 in the message)

A temp fix...is simply a reboot. The chances of the memory allocation lining up again the same way to cause the error is slim, but not entirely impossible. A fix is being worked on at the moment.

SBIE2303 Could not hook VerifyCatalogFile (888)
SBIE2318 DLL initialization failed for 'setupapi.dll'
SBIE2303 Could not hook WinStationOpenServerW (999)
SBIE2318 DLL initialization failed for 'winsta.dll'
SBIE2303 Could not hook NsiRpcRegisterChangeNotification (999)
SBIE2318 DLL initialization failed for 'winnsi.dll'
