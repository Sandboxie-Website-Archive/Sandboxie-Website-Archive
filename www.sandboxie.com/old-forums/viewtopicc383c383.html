BEFORE REPORTING A PROBLEM:

* Delete the contents of your sandbox and check if it makes a difference.

* Make sure you use the latest official version of Sandboxie or the latest beta version.

INCLUDE IN YOUR PROBLEM REPORT:

* The version number of Sandboxie that you use. In Sandboxie Control program, select Help menu, then click About Sandboxie.

* The version number of Windows that you use, and if it is a 64-bit version.

* Specify clear names for any third-party software that are involved. Be sure to include download links for such software.
