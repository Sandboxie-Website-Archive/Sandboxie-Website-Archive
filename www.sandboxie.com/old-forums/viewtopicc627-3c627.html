I use Start/Run Restrictions in several sandboxes. There are quite a few programs that I have allowed to run when I use Firefox for normal Internet usage, but a much smaller list for the sandbox that I use for banking purposes.
My software firewall is used for controlling Internet access.

I no longer use a Blocked Access setting in any of my sandboxes.
Blocked Access will result in an error message if some program tries to access the folder.

I prefer to use "Write-Only Access" because it completely hides the contents of the folder from sandboxed programs, and yet I can still set that folder as a Recovery Folder in case I do want to save something there.
I automatically include that setting for all of my sandboxes, except for one where I occasionally want to save individual emails to a sub-folder that's underneath the Documents folder.
