Show sandboxed files outside of sanbox

  • Quote

Post by Scott » Thu Feb 05, 2009 2:27 pm

It would be nice if you could see the files that were created in the sandbox out side of the sandbox.

Example: While running Firefox in the sandbox you download a file to the desktop. This file should appear on your desktop outside of the sandbox with a special icon so that you know it is a sandboxed file.
When you run the file it will automatically run in the sandbox.
If you clear the sandbox the file will disappear.


This would be a great feature to have so that it is easier to interact with programs that are installed into a sandbox. (the startmenu shortcuts and desktop shortcuts would be on your desktop)

Thanks for the great program!

-Scott
