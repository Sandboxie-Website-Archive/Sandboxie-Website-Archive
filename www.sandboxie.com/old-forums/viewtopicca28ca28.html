Bellzemos wrote: Also, when clicking on a DOC file in a sandboxed Windows Explorer the opening of the file takes a much longer time VS unsandboxed. I'm using paid Sandboxie 4.14 on Windows 7 x64.
Hi Bellzemos. I think we talk about this before. In my XP, I use MS Office. And don't use MS Office in W7. In my experience running Office in XP under version 4, I found disabling Drop rights makes Word run smooth. If I run Word files in a sandbox with Drop rights, I experience what you describe in the quote. So, I workaround the issue by not ticking Drop rights in sandboxes where I run Word files. I dont experience issues forcing Word or any other Office program. Or have to untick Drop rights for any other MS Office program but Word. For me, not enabling Drop rights in a couple of sandboxes is nothing.
