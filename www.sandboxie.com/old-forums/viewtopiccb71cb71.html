Hello Healt,

By recovering files, you are sending them outside of your Sandbox. Thus, any recovered file/application will open un-sandboxed. You can skip that [process by closing the Immediate recovery dialog.
Please see this entry for more information (I recommend you go thru the entire Getting Started tutorial as it contains very useful information):
https://www.sandboxie.com/GettingStartedPartFour

You can use the link I sent before in order to create a shortcut in Sandboxie OR, open Windows Explorer Sandboxed, create the shortcut there and launch it (anything launched from a Sandboxed Windows Explorer will open inside Sandboxie).

Hello Healt,

Here are some options that you can try:

- Create a shortcut using the steps I provided before:
https://www.sandboxie.com/ConfigureMenu#shell
Sandboxie Control ---> Configure ---> Windows Shell Integration
Click "Add Shortcut icons"
Follow the steps and navigate to the location of your game to create a shortcut that will only launch in the selected Sandbox.
[To find where your game is installed right-click on your game in Steam --> Properties ---> Local Files --> Browse Local Files... ]

- Use Windows Explorer Sandboxed (right-click on your Sandbox --> Run Sandboxed ---> Run Windows Explorer), navigate to the desired location and open the game that way.

- Launch Steam in Sandboxie, so that you can start the game directly from a sandboxed Steam session.

-Drag the non-sandboxed shortcut into Sandboxie control.
