Been a while so I figured I'd drop an update about my lingering programs issue. I rewrote my rules (well a lot of copy and pastes/alterations) removing the multiple templates entirely and instead using one large template for every single box with specific (!) exclusions and a few <ProcessGroups> pooled from multiple boxes in the GlobalSettings and I am no longer having any lingering sbie programs! I still find it odd that using multiple templates this way would cause anything to linger but at least it's been worked around and now my ini is a few KBs smaller as an added bonus!
