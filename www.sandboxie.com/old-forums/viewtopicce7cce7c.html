As for UAC, is there a difference between "completely disabled" and "never notify"? Otherwise I suspect it just runs applications at the user's privileges that started them. (Which by default is likely administrative.)
In Windows 7 when you set UAC to "Never notify" it forces any application to run in elevated privileges while in Windows 8 it wont, but when an application elevate itself it allows without notifying you.

Disabling UAC means to set UAC to "Never notify".

In Windows 7, a User Account that is without admin rights will not be affected when setting UAC to "Never notify".
