Hello n15,

Welcome to the forums.

In order to ensure applications are running inside the Sandbox, you can try this:
Open Sandboxie control, then go to File --> Is Window Sandboxed?.
Drag the Finder icon to your sandboxed application and it will tell you if it is running inside the Sandbox.

We have an FAQ section, as well as a Known Conflicts section that you will find useful:
https://www.sandboxie.com/index.php?HelpTopics

Usually, when there is a conflict you would notice something not working right. When all is well, sandboxed programs pretty much work as if you were not running them sandboxed. Not much difference. Using the browser for this example, sometimes if there is a conflict, when you try to run the browser, nothing runs or you might see a blank page. Another sign of a conflict would be if your browsers takes long to pop up running sandboxed after you click the icon. When you close the browser, if deleting the sandbox takes more than a few seconds that can also be a sign of a conflict.

But right now, I think you are doing good.

Bo
