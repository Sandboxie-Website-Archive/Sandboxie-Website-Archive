They are stored inside RegHive, at sandbox folder.

You can mount the file into registry to see the modifications or you can use a tool to do it.

The tool is named Buster Sandbox Analyzer and it includes a feature to see the registry keys and values of a RegHive file generated by Sandboxie.

Buster Sandbox Analyzer is available at http://bsa.isoftware.nl

While the sandbox is in use, you can also run the Registry Editor program unsandboxed (regedit.exe) and view the entries underneath HKEY_USERS\Sandbox_...
That's where Sandboxie mounts the Reghive file to the real Registry.
