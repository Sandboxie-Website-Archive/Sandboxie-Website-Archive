Post by Guest10 » Thu Oct 07, 2010 8:15 pm

I made a few local templates for Avant, if you want to try any of them.
I never did figure out exactly why "avant.exe" starts "ybrowser.exe", but at least one template needs the "ybrowser.exe" program in the setting - and the others do not seem to need "ybrowser.exe".
----------------------------
Based on my own computer (XP), it appears that Avant places its profile folder at:
%AppData%\Avant Profiles\.default

For Vista/Win 7, I think that this would correspond to:
C:\Users\(user)\AppData\Roaming\Avant\Avant Profiles\.default

For XP, it would be:
C:\Documents and Settings\(user)\Application Data\Avant Profiles\.default

If the Avant profile is not located there, the templates below will not work for you.
If the profile folder is not located as above, reply to this post and state where your Avant profile folder is actually located.
------------------------------
Add only the local templates that you will actually use, because unused local templates will cause a prompt from Control, to remove them if they are not actually in use.
Copy this entire post into a Notepad document, and Paste the template(s) when off line, as follows.
Sandbox Settings > Applications> Local > Create New
Paste the template's code into the window, one template at a time, and click OK in the "Create New" window.

After "Creating" all of the local templates you will be using, you will also need to "Add" them to the sandbox of choice at:
(sandbox name) > Sandbox Settings > Applications > Local > select and then "Add" each template - one template at a time.
When all have been added, click OK.
------------------------------
[Template_Local_Avant_Force]
Tmpl.Class=Local
Tmpl.Title=Force Avant to run in this sandbox (Registered version only)
ForceProcess=avant.exe

[Template_Local_Avant_Profile_DirectAccess]
Tmpl.Class=Local
Tmpl.Title=Allow direct access to entire Avant profile folder
Tmpl.Comment=This setting will save everything except cookies
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*

[Template_Local_Avant_Cookies]
Tmpl.Class=Local
Tmpl.Title=Allow direct access to Avant cookies
Tmpl.Comment=Avant saves cookies in IE's "Cookies" folder
OpenFilePath=ybrowser.exe,%Cookies%

[Template_Local_Avant_Bookmarks_DirectAccess]
Tmpl.Class=Local
Tmpl.Title=Allow direct access to Avant bookmarks
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\bookmarks.dat*

[Template_Local_Avant_Settings]
Tmpl.Class=Local
Tmpl.Title=Allow direct access to the Avant program configuration settings
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\conf.dat*
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\conf\conf.dat*

[Template_Local_Avant_Tab_Settings]
Tmpl.Class=Local
Tmpl.Title=Allow direct access to Avant saved tab settings and the "typed url's" list
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\closedtabs.dat
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\recenttabs.dat
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\reopen.dat
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\urls.dat

[Template_Local_Avant_FormData_RSS_DirectAccess]
Tmpl.Class=Local
Tmpl.Title=Allow direct access to Avant locally stored form data and RSS feeds
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\formdata\forms.dat*
OpenFilePath=avant.exe,%AppData%\Avant Profiles\*\rss\rss.dat*


[Edit]This post has been simplified, to remove the previous [TemplateSetting] instructions.
Therefore the templates, as listed above, will not be able to make use of Sandboxie's ability to specify an alternate profile folder location for Avant.
