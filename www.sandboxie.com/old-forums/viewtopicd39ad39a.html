LoluJaafJofe,

You can right-click on your Sandbox --> Delete contents to delete everything in it. Here's more information regarding the delete settings:
https://www.sandboxie.com/index.php?Del ... invocation

As for Windows Explorer, you can Right-click on your Sandbox ---> Run Sandbox ---> Run Windows explorer.
Another option is to Select View ---> Files and Folders and navigate from there and use the recovery features as needed (right- click on a file ---> Recover...).
Or, you can Right-Click --> Run Sandboxed to open the desired folder Sandboxed.
