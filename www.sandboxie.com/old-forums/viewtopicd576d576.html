All,

I was finally able to repro the issue in one VM, but several factors had to take place:
1 - Update from CU to FCU. [After this, I tested the previously installed Norton, no issues.]
2 - Have an expired Norton Security Trial.
3 - Download a new trial (this time Norton Security Deluxe (was the only one available)) .
4 - Allow Norton Security Deluxe to remove old Norton and install newer version - Reboot.
5 - Updated Sandboxie to latest beta (fully removed it, then created a new Sandbox).

After those steps, I am receiving SBIE2101 Object name not found: , error OpenProcess (C0000022) access=001FFFFF initialized=1 when I try to launch web browsers. So far, I have not been able to find a workaround but I made the devs aware of the issue.

I still need to test if re-installing Norton (fully removing it, then installing from scratch) makes a difference, since it was working fine on the previous fresh installation.
