Post by Unknown_User_605 » Sun Apr 08, 2007 6:43 am

Oh Yes - thats my problem. Thanks tzuk !
I wanted to open the directory D:\T and not everything like D:\T*
I modified in my sandboxie.ini all "OpenFilePath" Entries ending with a "\" eg, "D:\T\"
Now everything works fine - only this directory is opened.

I suggest making this feature clear understandable in the Online Help:

-------------------------------------------------------
...
[DefaultBox]
OpenFilePath=C:\Downloads
OpenFilePath=C:\*Downloads\MyFile.txt
OpenFilePath=*.eml
OpenFilePath=iexplore.exe,%Favorites% OpenFilePath=msimn.exe,*.eml

The first example setting specifies that any files (or folders) created in the folder C:\Downloads (and in any folder below it) will not be sandboxed.
...
--------------------------------------------------------

Exactly it has be be defined as "OpenFilePath=C:\Downloads\" to reach the described feature.
