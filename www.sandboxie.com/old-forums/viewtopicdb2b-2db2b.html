Re: 5.15 Beta Available (latest version 5.15.7)

Posted: Thu Dec 08, 2016 11:27 am
by Barb@Invincea
424tsiai,

Please disable the screen readers template, close all your Sandboxed apps, then do the following:

In SBIECtrl select Configure --> Edit Configuration
A Notepad file will open
Inside, you will find the [Global Settings], followed by the settings for each of your Sandboxes [DefaultBox] , etc...
Copy OpenWinClass=SWFlash_PlaceHolderX at the end of the [DefaultBox] area (Or at the end of any Sandbox that you want ).
Save, go back to SBie Control and select Configure --> Reload configuration
Test flash.

If you need additional assistance setting this up, do not hesitate to send me a PM.

Regards,
Barb.-
