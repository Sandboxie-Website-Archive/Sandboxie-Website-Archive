Posted: Fri Nov 27, 2015 5:15 pm

I use Sandboxie 5.06, Win7 x64

I have "D:\Temp" added among the Quick Recovery folders. When I place there an empty file, it is not offered in the Sandbox > Quick recovery dialog. Whenever the file has a content, it is offered there. The same happens with empty folders.

This behaviour may be a problem e.g. when one copies a whole directory tree that contains placeholder files. At this situation, you want get an exact copy of the directory tree - empty files and directories will disappear from the copy.
