Hi Todd, I understand your question, I also wondered about this a long time ago When you force your Downloads folder in C drive, the folder its not supposed to show the yellow border. But if you run a file that's inside the folder, then that file runs sandboxed and shows the yellow border.

There are times were the yellow border would show when you force a folder. For example, if you force your USB drives, as you insert a flash drive, your USB folder opens up showing the yellow border but that is because when you insert the flash drive the folder opens in a sandboxed version of Windows explorer. If you close this Window and later navigate back to your USB folder, you will not see the yellow border anymore. But files that run out of it, do. HTH.
