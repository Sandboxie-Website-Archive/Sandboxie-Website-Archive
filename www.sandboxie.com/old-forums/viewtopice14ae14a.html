Sandboxie version 2.55, 2.57, 2.59 Released for Testing

Posted: Fri Aug 18, 2006 12:02 pm
by tzuk
Changes:

- Delete Sandbox now takes only a couple of seconds, with the bulk of the operation done in the background

- Can run "Start.exe /box:SomeBox delete_sandbox" to invoke delete.

- Removed checkbox that would keep Internet Explorer AutoComplte information across deletes of sandbox.

- Added option to let sandboxed programs (most importantly Internet Explorer) store AutoComplete data outside the sandbox.

- Removed annoying pop up message boxes regarding ReportEvent and ControlService.

- Fixed bug in identification of default mail reader program.

- Fixed bug in passing Web links from sandboxed program to Firefox.

Update:

Version 2.59 is now available.
