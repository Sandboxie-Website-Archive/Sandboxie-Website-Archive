Post by rpljhun » Wed May 13, 2015 6:21 am

How about following these procedures:

1. Delete the contents of the sandbox you're using, then remove the sandbox.
2. Create New Sandbox (ie. maplestory - Copy Setting is None)
3. Run sandbox windows explorer(Sandbox Menu -> maplestory -> Run Sanboxed -> Run Windows Explorer)
4. Using the sandbox windows explorer, browse into the client installer folder and then run it(Double Clicked). (IMPORTANT: DON'T CLOSE THE SANDBOX WINDOWS EXPLORER)
5. After installation, run the game by going to Sandbox Menu -> maplestory -> Run Sanboxed -> Run From Start Menu look for the game launcher.

If you can't find the launcher in start menu. Use the sandbox windows explorer that you open a while ago, browse into the installation folder probably in C:\Program Files (x86)\maplestory... then look for the game launcher and run it(d-clicked)
