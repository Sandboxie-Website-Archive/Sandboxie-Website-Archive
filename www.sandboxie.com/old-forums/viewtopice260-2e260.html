Hello-

Installing games inside the Sandbox may or may not work. If the game installs drivers/services, those will not work inside Sandboxie (thus, you will have to install the game on your host and then run it Sandboxed).
If the game installation went thru, you will find the files inside the Container folder (typically C:\Sandbox\Username\SandboxName\Game-Path_Selected).
