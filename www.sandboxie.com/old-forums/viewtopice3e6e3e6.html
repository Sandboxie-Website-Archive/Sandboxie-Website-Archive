Applications that require Services to be installed my be blocked by SBIE.

Programs that require separate drivers simply will not work with SBIE. That's not possible at all.

Also, more and more online game hosts are flagging SBIE as a "cheat" program as well as it's being used for purposes it wasn't directly intended for.

Usually, the only other option is to install the software on your host and then right click and select "Run as sandboxed."
