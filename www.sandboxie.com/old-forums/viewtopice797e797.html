Hello BatmanCatman,

Still cannot repro. I have Media Player Classic playing a video loop + Chrome running in the same Sandbox...so far, so good.

As for your logs, there is a lot going on there, but I did notice something that did not come up on mine:
\Sessions\1\BaseNamedObjects\MediaPlayerClassicW

Try opening that path. Right-click on your Sandbox --> Sandbox Settings--> Resource Access--> IPC Access---> Direct Access
Paste:
*\BaseNamedObjects*\MediaPlayerClassicW
Hit Apply and Okay your way out.
Configure --> Reload Configuration
Re-try (if not difference, please remove this configuration).

If you can provide more info regarding the MPC version that you are using and type of video (avi, mp4, etc), I'll keep testing.
Also do try fully removing your security software (disabling it sometimes doesn't make a difference) to rule out any compatibility issues.
