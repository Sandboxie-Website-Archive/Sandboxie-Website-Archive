Hi All!

Just want to share my own way to make sandboxie portable.

This version supports plugins and settings storing from one run to another as well as configuring Sandbox folder place.

So here it is.

To start Sandboxie simply run start.cmd. No need to reboot after that. When your work is done please run stop.cmd - in this case all services will be stooped, but driver will remains in memory until next reboot (so please keep it in mind).

All initial settings are stored in Sandboxie.ini.template file. Please look thru it. As you can see it is simple sandboxie.ini file but with possibility to include in the path $(InstallDrive) template which means the drive where start.cmd started. Syou can simply change paths in the way you see fit including plugin paths, sandbox paths etc.

If you will change something during your work Sandboxie.ini.template will be updated in according to your changes.

It would be good to see some comments / suggestions here.
