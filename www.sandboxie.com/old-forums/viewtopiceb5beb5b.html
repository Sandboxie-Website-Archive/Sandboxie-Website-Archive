Colored border doesn't work on second monitor

Posted: Mon Feb 23, 2015 2:46 pm
by Special
Hi, using latest Sandboxie v4.16, (paid if that matters) and have noticed that when fullscreen applications are ran on a second monitor the colored border (usually yellow by default) does not display, on the primary monitor this works fine.

As a temporary fix I've found that dragging the sandboxed window around a bit moving it slightly off from center will cause the border to display.

On primary monitor it works great.
http://i.imgur.com/lrxzUB3.jpg


Secondary not so great.

Moving it some will bring it back.

http://i.imgur.com/HLUSWlV.jpg


Shrinking the window size also brings it back.
