by IronBunny
Make sure you have Comodo internet security/antivirus/firewall box ticked [+] in sandboxie control/configure/software compatibility.

Mine was unchecked even though i thought it was when i first got the prompt on sandboxie installer.

Also add the entire sandboxie folder to "detect shellcode injection" exclusion list. Located Defence+/Hips/Hips settings/under advanced.

Previously nothing would run. Not even an error. Put sandboxie folder into exception. I got an error at least. Then ticked compatibility and things seem to run no problem so far.
