I have Internet Explorer as my Default Browser.

I've set Firefox as the default browser inside one of my Sandboxes. So if I open an url shortcut inside of this sanbox, Firefox is opened by default.

But when I start a sandboxed web broser with "Run Sandboxed -> Run Web Broser" in this Sandbox, Internet Explorer starts instead of Firefox.

It would be nice to be able to set the default programs for each Sandboxes (or for all) for : Web Browser, Email Reader and even for Windows Explorer.

Posted: Wed Feb 25, 2009 4:37 pm
by tzuk
I know about this, the reason is (obviously) that "Run Web Browser" first checks what is the Web browser and then goes sandboxed. If it was the other way around, this would not be a problem. However, it is rarely a concern so I'm not sure I'll fix it. An easy workaround for you is to use Sandboxie Control > Configure menu > Windows Shell Integration > Add Shortcut Icons and create a shortcut to run your sandboxed Firefox. Then use this shortcut instead of "Run Web Browser".
