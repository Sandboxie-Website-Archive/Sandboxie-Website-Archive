Post by MichaleTron » Fri Aug 19, 2016 12:01 pm

The error: Update failed (error: 3)An error occurred while checking for updates: Update check failed to start (error code 3: 0x80080005 -- system level)
has occurred due to the Administrative privileges means your Google Updater were not running when you're running your Google Chrome, so you need first to go to your C:\Users\USERNAME\AppData\Local\Google\Update.

In this directory you will see GoogleUpdate.exe and then you need to run it as administartor.

Then finally Launch your Google Chrome browser then try to update your chrome. It will start updating automatically.

The original solution is published here.

You should always update your browser (or any software run as sandboxed) outside of the sandbox first.
Update. Then confirm it was updated.
Delete the contents of your sandbox.
Run the updated program in that sandbox.
