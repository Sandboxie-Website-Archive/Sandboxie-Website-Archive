Post by rpljhun » Thu Aug 11, 2016 12:18 pm

I think there is something wrong with your shortcut and you have pinned this shortcut as well. Check the shortcut properties, the shortcut should be

Code: Select all

"C:\Program Files\Sandboxie\SbieCtrl.exe" /open
If you omitted the "/open" the control window will not show.
Check also if run as minimized, it should be run as normal window.

And you should remove Norton. We have it as a conflict program for a reason.
It causes many issues you may have or will have. It will cause you headaches with SBIE.

This why you're flagged with a pop up box when SBIE is installed or when you run software compatibility from SBIE control.

It's listed here: http://www.sandboxie.com/index.php?KnownConflicts

We generally won't troubleshoot SBIE issues if a known conflict program is installed as 99% of the time it's related to that. Your case was just a shortcut issue...
