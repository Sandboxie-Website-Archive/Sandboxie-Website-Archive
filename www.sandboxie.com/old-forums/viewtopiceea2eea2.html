I'm trying to install an IME into a sandbox to try it out without success - I'm still interested in if it can be achieved but that's not what I'm trying to ask here right now.

During this process, I found that if I change the Language Bar setting inside the sandbox, it would affect my real OS.
  1. Open the "Text Services and Input Languages" window inside the default sandbox.
    Either open a sandboxed Windows Explorer and locate it in Control Panel, or run it directly:

    Code: Select all

    RunDll32.exe shell32.dll,Control_RunDLL C:\Windows\system32\input.dll
  2. Confirm that the "Text Services and Input Languages" window is running under sandbox supervision:
    [#] in title bar, yellow border, shows up within Sandboxie Control
  3. Switch to "Language Bar" tab on top, select "Floating On Desktop" / "Docked in the taskbar" then Apply setting
  4. The language bar in real OS is affected by the changes done by sandboxed "Text Services and Input Languages" window.
Isn't Sandboxie supposed to prevent that kind of change affecting the real OS? How can I stop that?
