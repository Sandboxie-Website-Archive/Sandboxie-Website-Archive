Re: Is there any chance Kaspersky 2016 gets supported? [SOLV

Post by Curt@invincea » Mon Aug 01, 2016 4:16 pm

Obviously there are people that want Sandboxie to work with Kaspersky. Kaspersky was working fine with Sandboxie until Kaspersky put out a release that takes over the Windows desktop object. Sandboxie must be able to sandbox the desktop object and it can't because Kaspersky has a hold on it.

I don't know if this is a bug or a design change by Kaspersky. But, I sent them 1 line of code that demonstrates the problem.

If it is a bug, they need to fix it. If it is truly part of their new design, then we have a problem. The only way for Sandboxie to work with that would be to un-sandbox the desktop object. Which weakens our security.

The bottom line is this: Kaspersky broke compatibility with Sandboxie, not the other way around.

There are many other A/V's that work just fine with Sandboxie. http://forums.sandboxie.com/phpBB3/view ... 11&t=21539. We have recommended Windows Defender not because it is the best A/V, but because it has never had an issue with Sandboxie and that is what we use here at Invincea.

Complaining here in our forum really doesn't accomplish anything.
