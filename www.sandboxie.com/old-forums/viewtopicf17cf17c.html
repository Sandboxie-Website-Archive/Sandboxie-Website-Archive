by Barb@Invincea
Hello Special ,

We test Fast Ring releases to see what's new and what may have broke. However, Fast Ring builds are usually unstable and present issues that are OS related, so it is hard to tell sometimes what's a Sandboxie issue and what's just a build issue. We unfortunately have to wait for the last builds (the ones that are release candidates) in order to properly test/troubleshoot/fix possible bugs.
On top of that, If the changes are too extensive, it can take a long time to find a way to fix them.

I hope that clarifies how this process works (at least from our end ).

We have created a dedicated windows 10 Fall's update thread explaining what's still not working, please have a look:
viewtopic.php?p=130657#p130657

Regards,
Barb.-
