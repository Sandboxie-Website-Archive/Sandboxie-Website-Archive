Cannot make Internet Shortcuts by dragging web links

Posted: Tue May 02, 2017 12:46 am
by TimW
I cannot drag web links (or the View Site Information icon in the address bar) from my browser to my desktop in order to create internet shortcuts to webpages. This problem applies to both sandboxed Chrome and Firefox.

This problem is not present with un-sandboxed browsers.
(I realize that I could use DeskCut for Firefox, which works nicely, even sandboxed. For Chrome there are also are menu-based methods which work, but these are not as convenient as the drag and drop method.)

Conditions:
- my sandbox is set for Direct Access to my desktop.
- browsers are sandboxed, but are not installed in a sandbox.
- using Windows 10, x64
- Chrome Version 58.0.3029.81 (64-bit) -up to date
- Firefox Version 53.0 (64-bit) -up to date
- AVG Internet Security
- Sandboxie Version 5.14 (64-bit)

- Thanks for any help

Re: Cannot make Internet Shortcuts by dragging web links

Posted: Tue May 02, 2017 12:34 pm
by Barb@Invincea
Hello TimW,

Drag-drop from the Sandbox to the Desktop does not work. However, you can open Windows Explorer Sandboxed and drag your files there.
Right-click on your Sandbox--> Run Windows Explorer
Navigate to your Desktop
Drag the urls from your Sandboxed browser to the Sandboxed Desktop.
Since you have direct access, they will show up on your host's desktop as well.

Regards,
Barb.-

TimW,

Sandboxed links dragged out don't work.

You can, however, drag or otherwise create one on the desktop outside of the sandbox then wrap a sandbox around them like any other link you want to run sandboxed. I do that with a couple of sites that are a little iffy but I want direct access from the desktop.

TimW, you can create the shortcut unsandboxed and place it in the Desktop, then convert it sandboxed via:

Configure>Windows shell integration, select the sandbox, and in the Desktop menu find the link and click it.

And leave the sandboxed icon in the Desktop.
