yxliang01 wrote: However, it is somewhat troublesome to execute them from outside. 1) Is there any way to make a shortcut?
Yes, you can make shortcuts.

Go to:
Sandboxie control>Configure>Windows Shell Integration, Click Add shortcut icons, Select the sandbox where you have installed the program, find the shortcut for the program in the Sandboxie Start menu that opens up (like under Desktop or Programs), Click it. After the click, youll find the sandboxed shortcut at the desktop.

You can also go into the folder of the sandboxed program via regular Windows explorer, right click the programs executable and select create shortcut, once is created you can drag it to the desktop.
