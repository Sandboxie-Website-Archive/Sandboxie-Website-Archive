  • Quote

Post by Curt@invincea » Thu Oct 01, 2015 5:13 pm

Combined 32/64 installer:
http://www.sandboxie.com/SandboxieInstall.exe

Separate:
http://www.sandboxie.com/SandboxieInstall32.exe
http://www.sandboxie.com/SandboxieInstall64.exe

Fixes in 5.06 RC
No code changes. Only the version # has been changed to 5.06.

Fixes in 5.05.3
1) Fixed several problems that could cause Chrome to crash in Windows 10.
2) Added logins.json for Firefox and derivatives to templates.ini regarding Firefox passwords

Fixes in 5.05.2
1) SBIE1222 Error with security token: [C0000061 / 33] caused by KB3088195 (Win 10 KB3097617) distributed on 10/13/2015 has been fixed. This also fixes BSOD when using Flash in Firefox.

Fixes in 5.05.1
1) Back by popular demand, the print spooler (spoolsv.exe) blocks file writes to the system temp folder (windows\temp) and the user temp folder (\Users\<username>\AppData\Local\Temp)
2) Fixed problem that sometimes caused IE 11 to immediately exit if there are 3rd party applications injecting into iexplore.exe with Sandboxie (like AV).
