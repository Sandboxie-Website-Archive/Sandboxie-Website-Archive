bjm wrote: How do I un-Hide...

Does Hide only Hide this specific 2335 or does Hide hide all 2335...?
1. Sandboxie Control > Configure > Edit Configuration (Sandboxie.ini is opened in notepad.exe)
2. Look for lines starting with SbieCtrl_HideMessage=
3. Delete the line you wish, save changes to Sandboxe.ini and your message boxes will pop up again.

Ok, when Sbie first blocks PDF Creator, a failed print job gets stuck in the PDF Creator print queue. From then on, PDF Creator will no longer work. After you dbl-click on SBIE1319 to allow the process to print to file, you must manually clear the print from the print queue. Then PDF Creator will function properly.
