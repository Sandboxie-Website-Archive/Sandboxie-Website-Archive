None of these versions of the Opera browser when running through Sandboxie does not work streaming audio on music sites,
such as:

www.beatport.com
music.yandex.ru
and so on (actually on any such sites!)

Once again, I note that the latest version of the Opera browser, which reproduces streaming audio on music sites without any problems when Opera itself is launched in Sanndboxie,
is the version of Opera 44.0.2510.1449.

But the most interesting is that. that without Sandboxie absolutely any version of Opera browser perfectly reproduces streaming audio on any music sites, which makes it absolutely possible to draw a conclusion about the problem in Sandboxie.
Dear Sandboxie developers, I understand that Chrome and Firefox are the priority for you, but please optimize Sandboxie's work with the latest versions of Opera browser when playing streaming audio on music sites.

Thank you!
