Post by Guest10 » Wed Mar 05, 2014 7:00 am

I would try Disabling all Firefox add-ons (when not sandboxed) and then see if the problem goes away when Firefox is run sandboxed.
Tools > Add-ons > Disable buttons

Make sure to set the sandbox to auto-delete before testing, since you don't want anything left in the sandbox that might cause a problem.
Sandbox Settings > Delete > Delete Invocation

There shouldn't be any reason for DDE to be needed by Firefox during those kind of operations, unless perhaps some add-on is trying to use it.
If you no longer see the problem with the add-ons disabled, start re-enabling some of them (when not sandboxed) to see which one is causing the problem.

There's a known problem with using DDE with Sandboxie v4, but as I said there shouldn't be any need for DDE when performing those operations in Firefox.
