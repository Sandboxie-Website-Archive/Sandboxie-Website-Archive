Changes in 5.19.5
1) Fixed Chrome - SBIE2101 Object name not found -(C0000022)

Changes in 5.19.4
1) Fixed error caused by MS update: SBIE2205 Service not implemented: CoUnmarshalInterface (W8)

Changes in 5.19.3
1) Fixed Chrome crash in Win 10.

Changes in 5.19.2
1) Fixed auto-recovery dialog failure to popup.

Changes in 5.19.1
1) Fixed problem with Firefox H264 video decoder.
2) Creation of IE embedded application objects are no longer blocked by default (which began in 5.18). See http://forums.sandboxie.com/phpBB3/view ... 11&t=24311
If IE is your default browser, it is being forced, and hyperlinks in MS Office are not working, you can use new option BlockIEEmbedding=y.
