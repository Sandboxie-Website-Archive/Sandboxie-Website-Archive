Sandboxie version 2.47 Released for Testing

Posted: Sat Jul 15, 2006 12:54 pm
by tzuk
New features since Version 2.45 beta 2

- Support for multiple sandboxes.

- Some protection against programs installing keyboard recorders, and programs trying to transmit fake keystrokes.

- Sandboxie Control can be easily configured to not start automatically.

- Support for Microsoft Office Outlook in addition to existing support for Microsoft Outlook Express.

- On fresh installs (not upgrade), the Sandbox driver will not be set to start automatically.

Changes and fixes:

- Sandboxed files copied to the clipboard in Sandboxie Explorer can be easily pasted outside the sandbox. Sandboxie Explorer can also be used to copy a file in one sandbox and paste it in another.

- Installer will correctly copy itself to the uninstall location, even if the name is changed from SandboxieInstaller.exe.

- Fixed a bug that would potentially crash systems with more than one CPU (also dual core or hyperthreaded CPU)
