Posted: Tue Dec 27, 2016 6:38 am
by Bertus
I use a couple of portable browsers. I have set up separate sandbox folders for given each browser.
I give certain permissions for each sandbox by using local templates (local application configuration). For certain browsers I like to enable or disable certain local templates and not delete them completely.
Each time I enable or disable a local template, Sandboxie asks if I want to delete the unused templates. I do not want to delete these templates so I have to click "No" each time I enable or disable (add or remove) a template.
Is there a way to disable this delete dialog box ?

greetz

Bertus wrote:Is there a way to disable this delete dialog box ?
There's no configuration setting for that, as far as I know.
You could Create a sandbox that you will never use and Apply the local templates to that sandbox.

Maybe give it a name that will place it, alphabetically, at the bottom of the list of your sandboxes.
