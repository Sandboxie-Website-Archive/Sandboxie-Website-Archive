Hi Grenpara. First of all, if I dont trust a program I wouldn't install it, or run it.

To protect your sensitive files and folders you can block sandboxed programs from accessing them via Sandbox settings>Resource access>File access>Blocked access, Click Add for navigating to the files and folders you want to protect.
