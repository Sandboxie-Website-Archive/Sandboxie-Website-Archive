Repro:

*Create sandbox (no settings changed)
*Open "Run Windows Explorer" from the sandbox's menu
*Leave sandboxed window open or close it (doesn't matter after it launches)
*Right-click desktop area (non-sandboxed) and choose "Personalize"
*Select "Background" from "Personalization" submenu.
*Select Picture background and a picture.
*The background won't change.
*At this point if you uninstall Sandboxie, the background will immediately change to the selected picture (using the installer to install the same version over the currently running version has the same effect).
*Restarting the computer also causes the previously selected background to display on logon.
